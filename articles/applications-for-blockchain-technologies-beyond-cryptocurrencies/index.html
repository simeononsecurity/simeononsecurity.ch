###дәҶи§ЈеҢәеқ—й“ҫжҠҖжңҜ

д»Җд№ҲжҳҜеҢәеқ—й“ҫпјҹ #

еҢәеқ—й“ҫжҳҜдёҖз§ҚеҲҶеёғејҸж•°жҚ®еә“пјҢе…Ғи®ёеӨҡж–№д»Ҙе®үе…ЁйҖҸжҳҺзҡ„ж–№ејҸйӘҢиҜҒж•°жҚ®е№¶е°Ҷж•°жҚ®ж·»еҠ еҲ°е…ұдә«иҙҰжң¬дёӯгҖӮеҢәеқ—й“ҫдёӯзҡ„жҜҸдёӘеҢәеқ—йғҪеҢ…еҗ«еүҚдёҖдёӘеҢәеқ—зҡ„еҠ еҜҶе“ҲеёҢеҖјпјҢд»ҺиҖҢеҲӣе»әдёҖдёӘдёҚеҸҜжӣҙж”№е’ҢйҳІзҜЎж”№зҡ„**еҢәеқ—й“ҫгҖӮиҝҷз§ҚжҠҖжңҜеҸҜе®һзҺ°дҝЎд»»пјҢеңЁи®ёеӨҡдәӨжҳ“дёӯж— йңҖдёӯд»ӢгҖӮ

еҢәеқ—й“ҫжҠҖжңҜзҡ„дё»иҰҒеҘҪеӨ„д№ӢдёҖжҳҜпјҢе®ғжҸҗдҫӣдәҶдёҖз§ҚеңЁеҸҜиғҪдә’дёҚзӣёиҜҶжҲ–дә’дёҚдҝЎд»»зҡ„еҗ„ж–№д№Ӣй—ҙе»әз«ӢдҝЎд»»зҡ„ж–№жі•гҖӮиҝҷжҳҜйҖҡиҝҮдҪҝз”ЁеҠ еҜҶз®—жі•е’Ңж•°еӯ—зӯҫеҗҚе®һзҺ°зҡ„пјҢеҸҜзЎ®дҝқдәӨжҳ“е®үе…Ёдё”дёҚеҸҜзҜЎж”№гҖӮеӣ жӯӨпјҢеҢәеқ—й“ҫжҠҖжңҜжңүеҸҜиғҪйқ©е‘ҪжҖ§ең°ж”№еҸҳжҲ‘д»¬иҝӣиЎҢдәӨжҳ“е’Ңд»·еҖјдәӨжҚўзҡ„ж–№ејҸгҖӮ

еҢәеқ—й“ҫеҰӮдҪ•е·ҘдҪңпјҹ #

еҢәеқ—й“ҫжҠҖжңҜдҫқйқ з”ЁжҲ·д№Ӣй—ҙзҡ„е…ұиҜҶжңәеҲ¶жқҘйӘҢиҜҒдәӨжҳ“е№¶е°ҶеҢәеқ—ж·»еҠ еҲ°й“ҫдёӯгҖӮеңЁ**еҺ»дёӯеҝғеҢ–зҡ„еҢәеқ—й“ҫзҪ‘з»ңдёӯпјҢжҜҸдёӘиҠӮзӮ№йғҪжңүдёҖд»ҪиҙҰжң¬пјҢе№¶зӢ¬з«ӢйӘҢиҜҒдәӨжҳ“гҖӮдәӨжҳ“йҖҡиҝҮдёҖдёӘз§°дёә “жҢ–зҹҝ”пјҲminingпјүзҡ„иҝҮзЁӢиў«жҚҶз»‘жҲҗеҢәеқ—пјҢе№¶ж·»еҠ еҲ°й“ҫдёҠгҖӮ

еҢәеқ—дёҖж—Ұж·»еҠ еҲ°еҢәеқ—й“ҫдёӯпјҢе°ұж— жі•жӣҙж”№жҲ–еҲ йҷӨгҖӮиҝҷдҪҝеҫ—еҢәеқ—й“ҫжҲҗдёәзҪ‘з»ңдёҠжүҖжңүдәӨжҳ“зҡ„*дёҚеҸҜжӣҙж”№зҡ„и®°еҪ•гҖӮеҠ еҜҶз®—жі•е’Ңж•°еӯ—зӯҫеҗҚзҡ„дҪҝз”ЁзЎ®дҝқдәҶеҢәеқ—й“ҫдёҠзҡ„ж•°жҚ®е®үе…ЁпјҢж— жі•зҜЎж”№гҖӮ

еҢәеқ—й“ҫжҠҖжңҜзҡ„дё»иҰҒзү№зӮ№ #

еҢәеқ—й“ҫжҠҖжңҜзҡ„дё»иҰҒзү№зӮ№еҢ…жӢ¬еҺ»дёӯеҝғеҢ–гҖҒдёҚеҸҳжҖ§гҖҒйҖҸжҳҺеәҰе’Ңе®үе…ЁжҖ§гҖӮеҺ»дёӯеҝғеҢ–жҳҜжҢҮжІЎжңүжҺ§еҲ¶зҪ‘з»ңзҡ„дёӯеӨ®жңәжһ„жҲ–дёӯд»ӢгҖӮиҝҷж„Ҹе‘ізқҖжІЎжңүдёҖдёӘе®һдҪ“еҸҜд»ҘжҺ§еҲ¶еҢәеқ—й“ҫдёҠзҡ„ж•°жҚ®пјҢдҪҝе…¶жҲҗдёәдёҖдёӘжӣҙеҠ ж°‘дё»е’ҢйҖҸжҳҺзҡ„зі»з»ҹгҖӮ

еҸҜзҜЎж”№жҖ§зЎ®дҝқж•°жҚ®дёҖж—Ұи®°еҪ•еңЁеҢәеқ—й“ҫдёҠпјҢе°ұж— жі•жӣҙж”№жҲ–еҲ йҷӨгҖӮиҝҷдҪҝеҫ—еҢәеқ—й“ҫжҲҗдёәзҪ‘з»ңдёҠжүҖжңүдәӨжҳ“зҡ„еҸҜйқ гҖҒйҳІзҜЎж”№и®°еҪ•гҖӮ

йҖҸжҳҺж„Ҹе‘ізқҖжүҖжңүеҸӮдёҺиҖ…йғҪиғҪиҺ·еҸ–зӣёеҗҢзҡ„дҝЎжҒҜгҖӮиҝҷе°ұеҲӣйҖ дәҶдёҖдёӘжӣҙеҠ ејҖж”ҫе’ҢеҚҸдҪңзҡ„зҺҜеўғпјҢеҗ„ж–№йғҪиғҪзңӢеҲ°е№¶йӘҢиҜҒзӣёеҗҢзҡ„ж•°жҚ®гҖӮ

йҖҡиҝҮдҪҝз”ЁеҠ еҜҶз®—жі•е’Ңж•°еӯ—зӯҫеҗҚжқҘзЎ®дҝқе®үе…ЁжҖ§гҖӮиҝҷдәӣжҠҖжңҜдҪҝд»»дҪ•дәәеҮ д№ҺдёҚеҸҜиғҪзҜЎж”№еҢәеқ—й“ҫдёҠзҡ„ж•°жҚ®пјҢзЎ®дҝқжүҖжңүдәӨжҳ“е®үе…ЁеҸҜйқ гҖӮ

е…¬е…ұеҢәеқ—й“ҫдёҺз§ҒдәәеҢәеқ—й“ҫ #

еҢәеқ—й“ҫзҪ‘з»ңжңүдёӨз§Қзұ»еһӢ–е…¬е…ұе’Ңз§ҒдәәгҖӮе…¬е…ұеҢәеқ—й“ҫпјҲеҰӮжҜ”зү№еёҒе’Ңд»ҘеӨӘеқҠпјүеҜ№д»»дҪ•жғіеҸӮдёҺзҡ„дәәејҖж”ҫпјҢд»»дҪ•дәәйғҪеҸҜд»ҘиҜ»еҸ–гҖҒеҶҷе…Ҙе’Ңжү§иЎҢжҷәиғҪеҗҲзәҰгҖӮиҝҷдәӣзҪ‘з»ңйҖҡеёёз”ЁдәҺеҺ»дёӯеҝғеҢ–еә”з”Ёе’ҢеҠ еҜҶиҙ§еёҒгҖӮ

еҸҰдёҖж–№йқўпјҢз§ҒжңүеҢәеқ—й“ҫз”ұзү№е®ҡзҡ„еҸӮдёҺиҖ…з»ҙжҠӨпјҢе…¬дј—ж— жі•и®ҝй—®гҖӮиҝҷдәӣзҪ‘з»ңйҖҡеёёз”ЁдәҺйңҖиҰҒжӣҙй«ҳзҡ„йҡҗз§Ғе’ҢжҺ§еҲ¶зҡ„дјҒдёҡи§ЈеҶіж–№жЎҲгҖӮз§ҒдәәеҢәеқ—й“ҫеҸҜж №жҚ®дјҒдёҡзҡ„е…·дҪ“йңҖжұӮиҝӣиЎҢе®ҡеҲ¶пјҢе№¶еҸҜз”ЁдәҺз®ҖеҢ–дёҡеҠЎжөҒзЁӢе’ҢйҷҚдҪҺжҲҗжң¬гҖӮ

жҖ»дҪ“иҖҢиЁҖпјҢеҢәеқ—й“ҫжҠҖжңҜжңүеҸҜиғҪеҪ»еә•ж”№еҸҳжҲ‘д»¬иҝӣиЎҢдәӨжҳ“е’Ңд»·еҖјдәӨжҚўзҡ„ж–№ејҸгҖӮеҢәеқ—й“ҫжҠҖжңҜе…·жңүеҺ»дёӯеҝғеҢ–гҖҒдёҚеҸҳжҖ§гҖҒйҖҸжҳҺжҖ§е’Ңе®үе…ЁжҖ§зӯүдё»иҰҒзү№зӮ№пјҢжҳҜдёҖз§ҚеҸҜйқ гҖҒе®үе…Ёзҡ„зі»з»ҹпјҢеҸҜе№ҝжіӣеә”з”ЁдәҺеҗ„з§ҚйўҶеҹҹгҖӮ

еҢәеқ—й“ҫеңЁйҮ‘иһҚйўҶеҹҹзҡ„еә”з”Ё #

еҢәеқ—й“ҫжҠҖжңҜеҪ»еә•ж”№еҸҳдәҶжҲ‘д»¬еҜ№йҮ‘иһҚзҡ„зңӢжі•гҖӮеҮӯеҖҹе…¶еҺ»дёӯеҝғеҢ–е’Ңе®үе…Ёзҡ„зү№жҖ§пјҢе®ғжңүеҸҜиғҪйў иҰҶдј з»ҹйҮ‘иһҚдҪ“зі»пјҢеёҰжқҘдёҖдёӘжӣҙеҠ йҖҸжҳҺе’Ңй«ҳж•Ҳзҡ„жңӘжқҘгҖӮ

жҷәиғҪеҗҲзәҰ #

жҷәиғҪеҗҲзәҰжҳҜеҢәеқ—й“ҫжҠҖжңҜжңҖд»Өдәәе…ҙеҘӢзҡ„еә”з”Ёд№ӢдёҖгҖӮе®ғд»¬жҳҜ**иҮӘеҠЁжү§иЎҢзҡ„и®Ўз®—жңәзЁӢеәҸпјҢеңЁж»Ўи¶ізү№е®ҡжқЎд»¶ж—¶иҮӘеҠЁжү§иЎҢеҗҲеҗҢжқЎж¬ҫгҖӮе®ғд»¬е»әз«ӢеңЁеҢәеқ—й“ҫжҠҖжңҜд№ӢдёҠпјҢеҸҜд»ҘеҲӣе»әеҺ»дёӯеҝғеҢ–зҡ„еә”з”ЁзЁӢеәҸгҖӮ

жҷәиғҪеҗҲзәҰеҸҜз”ЁдәҺиҮӘеҠЁеҢ–йҮ‘иһҚжөҒзЁӢпјҢеҰӮж”Ҝд»ҳгҖҒжё…з®—е’Ңз»“з®—д»ҘеҸҠдҝқйҷ©зҙўиө”гҖӮиҝҷеҸҜд»ҘеӨ§еӨ§еҮҸе°‘еҜ№дёӯд»Ӣзҡ„йңҖжұӮпјҢз®ҖеҢ–жөҒзЁӢпјҢдёәзӣёе…іеҗ„ж–№**иҠӮзңҒж—¶й—ҙе’ҢйҮ‘й’ұгҖӮ

еҲҶж•ЈејҸйҮ‘иһҚпјҲDeFiпјү #

еҺ»дёӯеҝғеҢ–йҮ‘иһҚпјҢжҲ–DeFiпјҢжҳҜжҢҮе»әз«ӢеңЁеҢәеқ—й“ҫжҠҖжңҜд№ӢдёҠзҡ„йҮ‘иһҚеә”з”ЁпјҢе…¶иҝҗдҪңж— йңҖдёӯд»ӢгҖӮиҝҷдәӣеә”з”ЁеҢ…жӢ¬еҖҹиҙ·гҖҒеҖҹж¬ҫгҖҒиө„дә§з®ЎзҗҶе’Ңдҝқйҷ©гҖӮ

DeFi дёәдј з»ҹйҮ‘иһҚжңәжһ„жңҚеҠЎдёҚи¶ізҡ„дёӘдәәжҸҗдҫӣдәҶжӣҙй«ҳзҡ„йҖҸжҳҺеәҰе’ҢиҺ·еҫ—йҮ‘иһҚжңҚеҠЎзҡ„жңәдјҡгҖӮз”ұдәҺз”ЁжҲ·еҜ№иҮӘе·ұзҡ„ж•°еӯ—й’ұеҢ…жӢҘжңүе®Ңе…Ёзҡ„жүҖжңүжқғе’ҢжҺ§еҲ¶жқғпјҢе®ғиҝҳе…Ғи®ёз”ЁжҲ·еҜ№иҮӘе·ұзҡ„иө„дә§иҝӣиЎҢжӣҙеӨ§зҡ„**жҺ§еҲ¶гҖӮ

и·Ёеўғж”Ҝд»ҳ #

еҢәеқ—й“ҫжҠҖжңҜеҸҜд»Ҙж¶ҲйҷӨдёӯй—ҙзҺҜиҠӮпјҢеҮҸе°‘дәӨжҳ“иҙ№з”Ёе’Ңз»“з®—ж—¶й—ҙпјҢд»ҺиҖҢдҝғиҝӣи·Ёеўғж”Ҝд»ҳгҖӮеҲ©з”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„ж”Ҝд»ҳзі»з»ҹпјҢдәӨжҳ“еҸҜд»Ҙе®һж—¶**з»“з®—пјҢиҖҢдё”жҜ”дј з»ҹж–№жі•жӣҙеҠ е®үе…ЁгҖӮ

иҝҷеҜ№йңҖиҰҒеҝ«йҖҹгҖҒе®үе…Ёең°и·ЁеўғжұҮж¬ҫзҡ„дёӘдәәе’ҢдјҒдёҡе°Өе…¶жңүеҲ©гҖӮе®ғиҝҳжңүеҠ©дәҺйҷҚдҪҺз”өжұҮе’Ңиҙ§еёҒе…‘жҚўзӯүдј з»ҹж”Ҝд»ҳж–№ејҸзҡ„зӣёе…іжҲҗжң¬гҖӮ

иө„дә§д»ӨзүҢеҢ– #

иө„дә§д»ЈеёҒеҢ–ж¶үеҸҠеңЁеҢәеқ—й“ҫдёҠе°ҶжҲҝең°дә§жҲ–иүәжңҜе“Ғзӯүе®һзү©иө„дә§иЎЁзӨәдёәж•°еӯ—д»ЈеёҒгҖӮиҝҷдәӣд»ЈеёҒеҸҜд»ҘеңЁж•°еӯ—еёӮеңәдёҠдәӨжҳ“пјҢдёәжҠ•иө„иҖ…жҸҗдҫӣеӨ§зҡ„жөҒеҠЁжҖ§е’ҢеҸҜи®ҝй—®жҖ§гҖӮ

иө„дә§д»ЈеёҒеҢ–иҝҳиғҪжҸҗй«ҳ**йҖҸжҳҺеәҰпјҢеҮҸе°‘ж¬әиҜҲиЎҢдёәпјҢеӣ дёәжүҖжңүдәӨжҳ“йғҪи®°еҪ•еңЁеҢәеқ—й“ҫдёҠпјҢеҫҲе®№жҳ“и·ҹиёӘе’ҢйӘҢиҜҒгҖӮиҝҷжңүеҠ©дәҺдёәеҸҜиғҪж— жі•дҪҝз”Ёдј з»ҹжҠ•иө„е·Ҙе…·зҡ„дёӘдәәејҖиҫҹж–°зҡ„жҠ•иө„жңәдјҡгҖӮ

жҖ»д№ӢпјҢеҢәеқ—й“ҫжҠҖжңҜжңүеҸҜиғҪд»ҘеӨҡз§Қж–№ејҸеҪ»еә•ж”№еҸҳйҮ‘иһҚдёҡгҖӮд»ҺжҷәиғҪеҗҲзәҰеҲ°еҺ»дёӯеҝғеҢ–йҮ‘иһҚпјҢд»Һи·Ёеўғж”Ҝд»ҳеҲ°иө„дә§д»ЈеёҒеҢ–пјҢеҸҜиғҪжҖ§жҳҜж— йҷҗзҡ„гҖӮйҡҸзқҖжҠҖжңҜзҡ„дёҚж–ӯеҸ‘еұ•пјҢжҲ‘д»¬жңүжңӣеңЁжңӘжқҘеҮ е№ҙзңӢеҲ°жӣҙеӨҡжҝҖеҠЁдәәеҝғзҡ„еҸ‘еұ•гҖӮ

еҢәеқ—й“ҫеңЁдҫӣеә”й“ҫз®ЎзҗҶдёӯзҡ„еә”з”Ё #

жқҘжәҗе’ҢеҸҜиҝҪжәҜжҖ§ #

еҢәеқ—й“ҫжҠҖжңҜиғҪеӨҹжҸҗдҫӣй«ҳзҡ„йҖҸжҳҺеәҰе’Ңиҙ§зү©зҡ„еҸҜиҝҪжәҜжҖ§пјҢеӣ жӯӨеңЁдҫӣеә”й“ҫз®ЎзҗҶиЎҢдёҡи¶ҠжқҘи¶ҠеҸ—ж¬ўиҝҺгҖӮйҖҡиҝҮеңЁеҢәеқ—й“ҫдёҠи®°еҪ•иҙ§зү©жөҒеҠЁжғ…еҶөпјҢе…¬еҸёеҸҜд»ҘзЎ®дҝқдә§е“Ғзҡ„зңҹе®һжҖ§е’ҢиҙЁйҮҸпјҢйҷҚдҪҺйЈҺйҷ©пјҢеҮҸе°‘еҒҮеҶ’дјӘеҠЈгҖӮиҝҷеҜ№дәҺйЈҹе“Ғе’ҢиҚҜе“ҒзӯүиЎҢдёҡе°ӨдёәйҮҚиҰҒпјҢеӣ дёәиҝҷдәӣиЎҢдёҡеҜ№дә§е“Ғзҡ„зңҹе®һжҖ§е’Ңе®үе…ЁжҖ§иҰҒжұӮжһҒй«ҳгҖӮ

дҫӢеҰӮпјҢеңЁйЈҹе“ҒиЎҢдёҡпјҢеҢәеқ—й“ҫжҠҖжңҜеҸҜз”ЁдәҺи·ҹиёӘеҶңдә§е“Ғд»ҺеҶңеңәеҲ°и¶…еёӮиҙ§жһ¶зҡ„жқҘжәҗгҖӮиҝҷдҪҝж¶Ҳиҙ№иҖ…иғҪеӨҹдәҶи§ЈйЈҹе“Ғзҡ„жқҘжәҗпјҢе№¶зЎ®дҝқйЈҹе“ҒеңЁж•ҙдёӘдҫӣеә”й“ҫдёӯеҫ—еҲ°е®үе…ЁгҖҒеҗҲд№ҺйҒ“еҫ·зҡ„еӨ„зҗҶгҖӮеҗҢж ·пјҢеңЁеҲ¶иҚҜиЎҢдёҡпјҢеҢәеқ—й“ҫжҠҖжңҜеҸҜз”ЁдәҺи·ҹиёӘиҚҜе“Ғд»ҺеҲ¶йҖ е•ҶеҲ°жӮЈиҖ…зҡ„жөҒйҖҡиҝҮзЁӢпјҢзЎ®дҝқиҚҜе“ҒдёҚжҳҜеҒҮеҶ’дјӘеҠЈдә§е“ҒпјҢе№¶еҫ—еҲ°еҰҘе–„еӮЁеӯҳгҖӮ

з®ҖеҢ–зү©жөҒ #

еңЁдҫӣеә”й“ҫз®ЎзҗҶдёӯдҪҝз”ЁеҢәеқ—й“ҫжҠҖжңҜзҡ„еҸҰдёҖдёӘеҘҪеӨ„жҳҜжңүеҸҜиғҪжҸҗй«ҳзү©жөҒзҡ„ж•ҲзҺҮе№¶йҷҚдҪҺжҲҗжң¬гҖӮйҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„дҫӣеә”й“ҫз®ЎзҗҶзі»з»ҹпјҢе…¬еҸёеҸҜд»ҘдјҳеҢ–иҝҗиҗҘпјҢеҮҸе°‘еҜ№дёӯй—ҙе•Ҷзҡ„йңҖжұӮгҖӮиҝҷеҸҜд»ҘдҪҝиҙ§зү©дәӨд»ҳжӣҙеҝ«гҖҒжӣҙй«ҳж•ҲпјҢе№¶еҮҸе°‘ж–Үд№Ұе·ҘдҪңе’ҢиЎҢж”ҝжҲҗжң¬гҖӮ

дҫӢеҰӮпјҢе…¬еҸёеҸҜд»ҘеҲ©з”ЁеҢәеқ—й“ҫжҠҖжңҜеңЁжө·е…іиҮӘеҠЁйӘҢиҜҒиҙ§зү©зҡ„зңҹе®һжҖ§гҖӮиҝҷеҸҜд»ҘеӨ§еӨ§еҮҸе°‘йӘҢиҜҒиҝҮзЁӢжүҖйңҖзҡ„ж—¶й—ҙе’ҢжҲҗжң¬пјҢе№¶йҷҚдҪҺж¬әиҜҲе’ҢдјӘйҖ зҡ„йЈҺйҷ©гҖӮ

еҮҸе°‘еҒҮеҶ’е•Ҷе“Ғ #

еҒҮеҶ’е•Ҷе“ҒжҳҜе…ЁзҗғеёӮеңәзҡ„дёҖдёӘдё»иҰҒй—®йўҳпјҢеҜ№ж¶Ҳиҙ№иҖ…е’ҢдјҒдёҡйғҪжңүеҪұе“ҚгҖӮеҢәеқ—й“ҫжҠҖжңҜеҸҜз”ЁдәҺйӘҢиҜҒдә§е“Ғзҡ„жқҘжәҗе’Ңзңҹе®һжҖ§пјҢеҮҸе°‘еҒҮеҶ’е•Ҷе“Ғзҡ„еҮәзҺ°гҖӮиҝҷеҸҜд»ҘйҖҡиҝҮеңЁеҢәеқ—й“ҫдёҠи®°еҪ•е•Ҷе“Ғзҡ„жөҒеҠЁжқҘе®һзҺ°пјҢд»ҺиҖҢдҪҝж¶Ҳиҙ№иҖ…е’ҢдјҒдёҡиғҪеӨҹеңЁдҫӣеә”й“ҫзҡ„д»»дҪ•зҺҜиҠӮйӘҢиҜҒдә§е“Ғзҡ„зңҹдјӘгҖӮ

дҫӢеҰӮпјҢеҘўдҫҲе“ҒеҲ¶йҖ е•ҶеҸҜд»ҘдҪҝз”ЁеҢәеқ—й“ҫжҠҖжңҜи·ҹиёӘдә§е“Ғд»Һе·ҘеҺӮеҲ°е•Ҷеә—зҡ„жөҒеҠЁгҖӮз„¶еҗҺпјҢж¶Ҳиҙ№иҖ…еҸҜд»ҘдҪҝз”ЁжҷәиғҪжүӢжңәеә”з”ЁзЁӢеәҸжү«жҸҸдә§е“ҒдёҠзҡ„дәҢз»ҙз ҒпјҢйӘҢиҜҒе…¶зңҹдјӘгҖӮиҝҷеҸҜд»ҘеӨ§еӨ§еҮҸе°‘еҒҮеҶ’е•Ҷе“Ғзҡ„еҮәзҺ°пјҢиҖҢеҒҮеҶ’е•Ҷе“ҒдјҡжҚҹе®ідјҒдёҡзҡ„еЈ°иӘүе’Ңж”¶е…ҘгҖӮ

еҸҜжҢҒз»ӯдҫӣеә”й“ҫ #

еҢәеқ—й“ҫжҠҖжңҜиҝҳеҸҜд»ҘйҖҡиҝҮи·ҹиёӘж•ҙдёӘдҫӣеә”й“ҫдёӯдә§е“ҒеҜ№зҺҜеўғзҡ„еҪұе“ҚпјҢеё®еҠ©дјҒдёҡеҲӣе»әжӣҙе…·еҸҜжҢҒз»ӯжҖ§зҡ„дҫӣеә”й“ҫгҖӮйҖҡиҝҮеңЁеҢәеқ—й“ҫдёҠи®°еҪ•ж•°жҚ®пјҢдјҒдёҡеҸҜд»ҘзЎ®е®ҡйңҖиҰҒж”№иҝӣзҡ„йўҶеҹҹпјҢе№¶е®һж–Ҫжӣҙе…·еҸҜжҢҒз»ӯжҖ§зҡ„еҒҡжі•гҖӮ

дҫӢеҰӮпјҢдёҖе®¶жңҚиЈ…еҲ¶йҖ е•ҶеҸҜд»ҘеҲ©з”ЁеҢәеқ—й“ҫжҠҖжңҜиҝҪиёӘжЈүиҠұжҲ–зҫҠжҜӣзӯүеҺҹжқҗж–ҷзҡ„жқҘжәҗгҖӮиҝҷж ·пјҢиҜҘе…¬еҸёе°ұиғҪзЎ®дҝқе…¶жқҗж–ҷзҡ„жқҘжәҗз¬ҰеҗҲйҒ“еҫ·ж ҮеҮҶпјҢе№¶дё”жҳҜеҸҜжҢҒз»ӯзҡ„гҖӮжӯӨеӨ–пјҢиҜҘе…¬еҸёиҝҳеҸҜд»Ҙи·ҹиёӘдёҺдә§е“Ғз”ҹдә§е’Ңиҝҗиҫ“зӣёе…ізҡ„иғҪжәҗж¶ҲиҖ—е’ҢзўіжҺ’ж”ҫгҖӮиҝҷж ·пјҢе…¬еҸёе°ұиғҪжүҫеҮәйңҖиҰҒж”№иҝӣзҡ„ең°ж–№пјҢе№¶е®һж–Ҫжӣҙе…·еҸҜжҢҒз»ӯжҖ§зҡ„еҒҡжі•пјҢеҰӮдҪҝз”ЁеҸҜеҶҚз”ҹиғҪжәҗжҲ–дјҳеҢ–иҝҗиҫ“и·ҜзәҝгҖӮ

з»“и®ә #

жҖ»д№ӢпјҢеҢәеқ—й“ҫжҠҖжңҜжңүеҸҜиғҪйҖҡиҝҮжҸҗдҫӣй«ҳзҡ„йҖҸжҳҺеәҰгҖҒеҸҜиҝҪжәҜжҖ§е’Ңж•ҲзҺҮпјҢеҪ»еә•ж”№еҸҳдҫӣеә”й“ҫз®ЎзҗҶгҖӮйҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„дҫӣеә”й“ҫз®ЎзҗҶзі»з»ҹ**пјҢе…¬еҸёеҸҜд»ҘзЎ®дҝқдә§е“Ғзҡ„зңҹе®һжҖ§е’ҢиҙЁйҮҸпјҢйҷҚдҪҺжҲҗжң¬пјҢе№¶еҲӣе»әжӣҙе…·еҸҜжҢҒз»ӯжҖ§зҡ„дҫӣеә”й“ҫгҖӮйҡҸзқҖеҢәеқ—й“ҫжҠҖжңҜзҡ„дёҚж–ӯеҸ‘еұ•пјҢжҲ‘д»¬еҫҲеҸҜиғҪдјҡзңӢеҲ°и¶ҠжқҘи¶ҠеӨҡзҡ„е…¬еҸёйҮҮз”Ёиҝҷз§ҚжҠҖжңҜжқҘеҠ ејәдҫӣеә”й“ҫз®ЎзҗҶжөҒзЁӢгҖӮ

еҢәеқ—й“ҫеңЁеҢ»з–—дҝқеҒҘйўҶеҹҹзҡ„еә”з”Ё #

е®үе…Ёзҡ„жӮЈиҖ…ж•°жҚ®з®ЎзҗҶ #

еҢәеқ—й“ҫжҠҖжңҜеҸҜдёәеӯҳеӮЁе’Ңз®ЎзҗҶжӮЈиҖ…ж•°жҚ®жҸҗдҫӣдёҖдёӘе®үе…ЁгҖҒйҖҸжҳҺзҡ„зі»з»ҹпјҢд»ҺиҖҢеҪ»еә•ж”№еҸҳеҢ»з–—дҝқеҒҘиЎҢдёҡгҖӮжңүдәҶеҹәдәҺеҢәеқ—й“ҫзҡ„еҒҘеә·дҝЎжҒҜдәӨжҚўпјҢжӮЈиҖ…еҸҜд»ҘжӣҙеҘҪең°жҺ§еҲ¶иҮӘе·ұзҡ„ж•°жҚ®пјҢеҢ»з–—жңҚеҠЎжҸҗдҫӣиҖ…д№ҹеҸҜд»Ҙе®үе…Ёй«ҳж•Ҳең°и®ҝй—®дҝЎжҒҜгҖӮиҝҷйЎ№жҠҖжңҜдҪҝдҝқйҷ©е…¬еҸёе’Ңж”ҝеәңжңәжһ„зӯүдёӯд»Ӣжңәжһ„дёҚеҶҚйңҖиҰҒз®ЎзҗҶжӮЈиҖ…ж•°жҚ®пјҢд»ҺиҖҢж”№е–„дәҶжӮЈиҖ…зҡ„жІ»з–—ж•ҲжһңгҖӮ

жӯӨеӨ–пјҢеҢәеқ—й“ҫжҠҖжңҜеҸҜзЎ®дҝқжӮЈиҖ…ж•°жҚ®йҳІзҜЎж”№пјҢжңӘз»ҸеҗҢж„ҸдёҚеҫ—зҜЎж”№жҲ–еҲ йҷӨгҖӮиҝҷж„Ҹе‘ізқҖжӮЈиҖ…еҸҜд»ҘзӣёдҝЎд»–д»¬зҡ„ж•°жҚ®жҳҜеҮҶзЎ®е’ҢжңҖж–°зҡ„пјҢиҝҷеҜ№дәҺеҒҡеҮәжҳҺжҷәзҡ„еҢ»з–—еҶізӯ–иҮіе…ійҮҚиҰҒгҖӮжӯӨеӨ–пјҢдҪҝз”ЁеҢәеқ—й“ҫжҠҖжңҜеҸҜд»ҘйҷҚдҪҺз®ЎзҗҶжҲҗжң¬пјҢз®ҖеҢ–жөҒзЁӢпјҢд»ҺиҖҢжҸҗй«ҳеҢ»з–—зі»з»ҹзҡ„ж•ҲзҺҮгҖӮ

иҚҜе“ҒеҸҜиҝҪжәҜжҖ§ #

дҪҝз”ЁеҒҮеҶ’иҚҜе“Ғе’ҢеҢ»з–—еҷЁжў°жҳҜеҢ»з–—дҝқеҒҘиЎҢдёҡзҡ„дёҖеӨ§йҡҗжӮЈгҖӮиҝҷдәӣдә§е“ҒдјҡеҜ№жӮЈиҖ…е®үе…Ёжһ„жҲҗйҮҚеӨ§еЁҒиғҒпјҢе№¶еҸҜиғҪеҜјиҮҙдёҘйҮҚзҡ„еҒҘеә·е№¶еҸ‘з—ҮгҖӮеҢәеқ—й“ҫжҠҖжңҜеҸҜз”ЁдәҺи·ҹиёӘиҚҜе“Ғе’ҢеҢ»з–—еҷЁжў°еңЁж•ҙдёӘдҫӣеә”й“ҫдёӯзҡ„жөҒеҠЁжғ…еҶөпјҢзЎ®дҝқдә§е“Ғзҡ„зңҹе®һжҖ§е’ҢиҙЁйҮҸгҖӮ

йҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„зі»з»ҹпјҢеҲ¶йҖ е•ҶгҖҒеҲҶй”Җе•Ҷе’ҢеҢ»з–—дҝқеҒҘжҸҗдҫӣе•ҶеҸҜд»Ҙи·ҹиёӘиҚҜе“Ғе’ҢеҢ»з–—еҷЁжў°д»ҺеҺҹдә§ең°еҲ°ж¶Ҳиҙ№ең°зҡ„жөҒеҠЁжғ…еҶөгҖӮиҝҷеҸҜзЎ®дҝқжӮЈиҖ…иҺ·еҫ—иҙ§зңҹд»·е®һзҡ„дјҳиҙЁдә§е“ҒпјҢеҜ№ж”№е–„жӮЈиҖ…зҡ„жІ»з–—ж•ҲжһңиҮіе…ійҮҚиҰҒгҖӮжӯӨеӨ–пјҢеҢәеқ—й“ҫжҠҖжңҜиҝҳиғҪеҮҸе°‘дёҺиҝҪиёӘдә§е“Ғзӣёе…ізҡ„ж—¶й—ҙе’ҢжҲҗжң¬пјҢд»ҺиҖҢжҸҗй«ҳдҫӣеә”й“ҫзҡ„ж•ҲзҺҮгҖӮ

дёҙеәҠиҜ•йӘҢдёҺз ”з©¶ #

дёҙеәҠиҜ•йӘҢе’Ңз ”з©¶еҜ№дәҺејҖеҸ‘жІ»з–—еҗ„з§Қз–ҫз—…зҡ„ж–°з–—жі•е’Ңз–—жі•иҮіе…ійҮҚиҰҒгҖӮз„¶иҖҢпјҢиҝҷдәӣиҝҮзЁӢиҖ—ж—¶й•ҝгҖҒжҲҗжң¬й«ҳпјҢж•°жҚ®зҡ„еҮҶзЎ®жҖ§е’ҢиҙЁйҮҸд№ҹдјҡеҸ—еҲ°еҪұе“ҚгҖӮеҢәеқ—й“ҫжҠҖжңҜйҖҡиҝҮеңЁеҲҶеёғејҸеҲҶзұ»иҙҰдёҠи®°еҪ•ж•°жҚ®пјҢеҸҜд»ҘжҸҗй«ҳдёҙеәҠиҜ•йӘҢзҡ„ж•ҲзҺҮе’ҢйҖҸжҳҺеәҰгҖӮ

йҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„зі»з»ҹпјҢз ”з©¶дәәе‘ҳеҸҜд»ҘзЎ®дҝқж•°жҚ®зҡ„иҙЁйҮҸе’ҢеҮҶзЎ®жҖ§пјҢиҝҷеҜ№дәҺеҒҡеҮәжңүе…іжӮЈиҖ…жҠӨзҗҶзҡ„жҳҺжҷәеҶізӯ–иҮіе…ійҮҚиҰҒгҖӮжӯӨеӨ–пјҢеҢәеқ—й“ҫжҠҖжңҜиҝҳиғҪеҮҸе°‘дёҺж•°жҚ®ж”¶йӣҶе’ҢеҲҶжһҗзӣёе…ізҡ„ж—¶й—ҙе’ҢжҲҗжң¬пјҢд»ҺиҖҢжҸҗй«ҳдёҙеәҠиҜ•йӘҢзҡ„ж•ҲзҺҮгҖӮиҝҷеҸҜд»ҘеҠ еҝ«ж–°з–—жі•е’Ңж–°з–—жі•зҡ„ејҖеҸ‘пјҢжңҖз»Ҳж”№е–„жӮЈиҖ…зҡ„жІ»з–—ж•ҲжһңгҖӮ

иҝңзЁӢеҢ»з–—е’ҢиҝңзЁӢзӣ‘жҺ§ #

иҝңзЁӢеҢ»з–—е’ҢиҝңзЁӢзӣ‘жҺ§еңЁеҢ»з–—дҝқеҒҘиЎҢдёҡи¶ҠжқҘи¶ҠеҸ—ж¬ўиҝҺпјҢеӣ дёәе®ғд»¬еҸҜд»Ҙи®©жӮЈиҖ…еңЁдё–з•Ңд»»дҪ•ең°ж–№иҺ·еҫ—еҢ»з–—дҝқеҒҘжңҚеҠЎгҖӮз„¶иҖҢпјҢиҝҷдәӣжңҚеҠЎе®№жҳ“еҸ—еҲ°е®үе…ЁжјҸжҙһе’Ңж•°жҚ®зӣ—зӘғзҡ„еҪұе“ҚгҖӮеҢәеқ—й“ҫжҠҖжңҜеҸҜд»ҘеңЁжӮЈиҖ…е’ҢеҢ»з–—жңҚеҠЎжҸҗдҫӣиҖ…д№Ӣй—ҙжҸҗдҫӣе®үе…Ёе’Ңй«ҳж•Ҳзҡ„йҖҡдҝЎпјҢд»ҺиҖҢеҠ ејәиҝңзЁӢеҢ»з–—е’ҢиҝңзЁӢзӣ‘жҺ§гҖӮ

йҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„зі»з»ҹпјҢжӮЈиҖ…еҸҜд»ҘзӣёдҝЎд»–д»¬зҡ„ж•°жҚ®жҳҜе®үе…Ёзҡ„пјҢжңӘз»ҸжҺҲжқғж–№ж— жі•и®ҝй—®гҖӮиҝҷеҸҜд»ҘжҸҗй«ҳжӮЈиҖ…еҜ№еҢ»з–—жңҚеҠЎзҡ„дҝЎд»»еәҰе’Ңж»Ўж„ҸеәҰгҖӮжӯӨеӨ–пјҢеҢәеқ—й“ҫжҠҖжңҜеҸҜеҮҸе°‘дёҺж•°жҚ®з®ЎзҗҶе’ҢйҖҡдҝЎзӣёе…ізҡ„ж—¶й—ҙе’ҢжҲҗжң¬пјҢд»ҺиҖҢжҸҗй«ҳиҝңзЁӢеҢ»з–—е’ҢиҝңзЁӢзӣ‘жҺ§зҡ„ж•ҲзҺҮгҖӮ

жҖ»д№ӢпјҢеҢәеқ—й“ҫжҠҖжңҜйҖҡиҝҮжҸҗдҫӣе®үе…Ёе’Ңй«ҳж•Ҳзҡ„зі»з»ҹжқҘз®ЎзҗҶз—…дәәж•°жҚ®гҖҒи·ҹиёӘиҚҜзү©е’ҢеҢ»з–—и®ҫеӨҮзҡ„з§»еҠЁгҖҒжҸҗй«ҳдёҙеәҠиҜ•йӘҢе’Ңз ”з©¶зҡ„ж•ҲзҺҮе’ҢйҖҸжҳҺеәҰпјҢд»ҘеҸҠеҠ ејәиҝңзЁӢеҢ»з–—е’ҢиҝңзЁӢзӣ‘жҺ§пјҢд»ҺиҖҢжңүеҸҜиғҪеҪ»еә•ж”№еҸҳеҢ»з–—дҝқеҒҘиЎҢдёҡгҖӮйҡҸзқҖиҝҷйЎ№жҠҖжңҜзҡ„дёҚж–ӯеҸ‘еұ•пјҢеҢ»з–—жңҚеҠЎжҸҗдҫӣе•Ҷе’Ңжңәжһ„еҝ…йЎ»жҺўзҙўе…¶жҪңеңЁеә”з”Ёе№¶жҺҘеҸ—е…¶дјҳеҠҝгҖӮ

еҢәеқ—й“ҫеңЁиғҪжәҗе’Ңе…¬з”ЁдәӢдёҡдёӯзҡ„еә”з”Ё #

иғҪжәҗе’Ңе…¬з”ЁдәӢдёҡиЎҢдёҡжӯЈеңЁиҝ…йҖҹеҸ‘еұ•пјҢиҖҢеҢәеқ—й“ҫжҠҖжңҜеңЁиҝҷдёҖиҪ¬еҸҳдёӯеҸ‘жҢҘзқҖиҮіе…ійҮҚиҰҒзҡ„дҪңз”ЁгҖӮеҢәеқ—й“ҫжҠҖжңҜеҸҜз”ЁдәҺеҲӣе»әеҺ»дёӯеҝғеҢ–иғҪжәҗзҪ‘гҖҒе®һзҺ°зӮ№еҜ№зӮ№иғҪжәҗдәӨжҳ“гҖҒз®ҖеҢ–и®Ўиҙ№е’Ңж”Ҝд»ҳжөҒзЁӢпјҢд»ҘеҸҠи·ҹиёӘзўіжҺ’ж”ҫе’ҢеҸ‘ж”ҫзўідҝЎз”ЁйўқеәҰгҖӮи®©жҲ‘д»¬жқҘиҜҰз»ҶжҺўи®Ёиҝҷдәӣеә”з”ЁгҖӮ

еҲҶж•ЈејҸиғҪжәҗзҪ‘ #

еҺ»дёӯеҝғеҢ–иғҪжәҗзҪ‘жҳҜдёҖз§ҚеңЁжң¬ең°дә§з”ҹе’Ңж¶ҲиҖ—иғҪжәҗзҡ„еҲӣж–°ж–№ејҸпјҢеҸҜеҮҸе°‘еҜ№йӣҶдёӯејҸиғҪжәҗдҫӣеә”е•Ҷзҡ„дҫқиө–гҖӮйҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„еҫ®з”өзҪ‘пјҢзӨҫеҢәеҸҜд»ҘеңЁжң¬ең°дә§з”ҹе’Ңж¶ҲиҖ—иғҪжәҗпјҢеҮҸе°‘дј иҫ“жҚҹиҖ—е№¶жҸҗй«ҳж•ҲзҺҮгҖӮиҝҷиҝҳиғҪдёәж¶Ҳиҙ№иҖ…иҠӮзңҒ**жҲҗжң¬пјҢе№¶жҸҗдҫӣжӣҙеҸҜйқ зҡ„иғҪжәҗжқҘжәҗгҖӮ

дҫӢеҰӮпјҢеңЁзәҪзәҰеёғйІҒе…Ӣжһ—пјҢеҲ©з”ЁеҢәеқ—й“ҫжҠҖжңҜеҲӣе»әдәҶдёҖдёӘзӨҫеҢәеҫ®з”өзҪ‘пјҢд»Ҙе®һзҺ°еҜ№зӯүиғҪжәҗдәӨжҳ“гҖӮеҫ®з”өзҪ‘з”ұеӨӘйҳіиғҪз”өжұ жқҝе’Ңи“„з”өжұ дҫӣз”өпјҢеұ…ж°‘еҸҜд»ҘдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„е№іеҸ°д№°еҚ–иғҪжәҗгҖӮиҝҷдёәзӨҫеҢәеҲӣйҖ дәҶдёҖдёӘжӣҙе…·еҸҜжҢҒз»ӯжҖ§е’Ңеј№жҖ§зҡ„иғҪжәҗзі»з»ҹгҖӮ

зӮ№еҜ№зӮ№иғҪжәҗдәӨжҳ“ #

еҢәеқ—й“ҫжҠҖжңҜиҝҳеҸҜд»ҘдёәиғҪжәҗз”ҹдә§иҖ…е’Ңж¶Ҳиҙ№иҖ…еҲӣе»әдёҖдёӘеёӮеңәпјҢд»ҺиҖҢе®һзҺ°еҜ№зӯүиғҪжәҗдәӨжҳ“гҖӮйҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„зі»з»ҹпјҢдёӘдәәе’ҢдјҒдёҡеҸҜд»ҘеңЁдёҖдёӘйҖҸжҳҺгҖҒе®үе…Ёзҡ„е№іеҸ°дёҠд№°еҚ–иғҪжәҗгҖӮиҝҷеҸҜд»ҘжҸҗй«ҳиғҪжәҗиө„жәҗзҡ„дҪҝз”Ёж•ҲзҺҮпјҢеҮҸе°‘жё©е®Өж°”дҪ“жҺ’ж”ҫгҖӮ

дҫӢеҰӮпјҢжҫіеӨ§еҲ©дәҡеҲ©з”ЁеҢәеқ—й“ҫжҠҖжңҜдәҶдёҖдёӘзӮ№еҜ№зӮ№иғҪжәҗдәӨжҳ“е№іеҸ°гҖӮиҜҘе№іеҸ°е…Ғи®ёиЈ…жңүеӨӘйҳіиғҪз”өжұ жқҝзҡ„е®¶еәӯеҗ‘йӮ»еұ…еҮәе”®еӨҡдҪҷзҡ„иғҪжәҗпјҢд»ҺиҖҢеҲӣе»әдёҖдёӘжӣҙеҠ еҸҜжҢҒз»ӯе’Ң**еҲҶж•Јзҡ„иғҪжәҗзі»з»ҹгҖӮ

жҷәиғҪи®ЎйҮҸе’Ңи®Ўиҙ№ #

йҖҡиҝҮдҪҝз”ЁжҷәиғҪеҗҲзәҰиҮӘеҠЁжү§иЎҢдәӨжҳ“пјҢеҢәеқ—й“ҫжҠҖжңҜеҸҜз®ҖеҢ–иғҪжәҗдҫӣеә”е•Ҷе’Ңж¶Ҳиҙ№иҖ…зҡ„и®Ўиҙ№е’Ңж”Ҝд»ҳжөҒзЁӢгҖӮиҝҷеҸҜд»ҘеҮҸиҪ»иғҪжәҗдҫӣеә”е•Ҷзҡ„з®ЎзҗҶиҙҹжӢ…пјҢжҸҗй«ҳж¶Ҳиҙ№иҖ…иҙҰеҚ•зҡ„еҮҶзЎ®жҖ§гҖӮжӯӨеӨ–пјҢеҹәдәҺеҢәеқ—й“ҫзҡ„зі»з»ҹеҸҜд»ҘжҸҗдҫӣиғҪжәҗдҪҝз”Ёжғ…еҶөзҡ„е®һж—¶ж•°жҚ®пјҢдҪҝж¶Ҳиҙ№иҖ…иғҪеӨҹе°ұе…¶иғҪжәҗж¶ҲиҖ—еҒҡеҮәжҳҺжҷәзҡ„еҶіе®ҡгҖӮ

дҫӢеҰӮпјҢиӢұеӣҪеҲӣе»әдәҶдёҖдёӘеҹәдәҺеҢәеқ—й“ҫзҡ„иғҪжәҗдәӨжҳ“е№іеҸ°пјҢд»Ҙе®һзҺ°еҜ№зӯүиғҪжәҗдәӨжҳ“гҖӮиҜҘе№іеҸ°дҪҝз”ЁжҷәиғҪеҗҲзәҰиҮӘеҠЁжү§иЎҢдәӨжҳ“пјҢе№¶жҸҗдҫӣе®һж—¶зҡ„иғҪжәҗдҪҝз”Ёж•°жҚ®гҖӮиҝҷдёәж¶Ҳиҙ№иҖ…еҲӣйҖ дәҶдёҖдёӘжӣҙеҠ йҖҸжҳҺе’Ңй«ҳж•Ҳзҡ„иғҪжәҗзі»з»ҹгҖӮ

зўідҝЎз”Ёйўқе’ҢжҺ’ж”ҫи·ҹиёӘ #

еҢәеқ—й“ҫжҠҖжңҜеҸҜдёәи·ҹиёӘзўіжҺ’ж”ҫе’ҢеҸ‘ж”ҫзўідҝЎз”ЁйўқеәҰжҸҗдҫӣдёҖдёӘйҖҸжҳҺгҖҒе®үе…Ёзҡ„зі»з»ҹгҖӮйҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„зі»з»ҹпјҢж”ҝеәңе’ҢдјҒдёҡеҸҜд»ҘжҝҖеҠұеҸҜжҢҒз»ӯеҒҡжі•пјҢеҮҸе°‘ж°”еҖҷеҸҳеҢ–зҡ„еҪұе“ҚгҖӮиҝҷд№ҹеҸҜд»ҘдёәеҮҸе°‘зўіжҺ’ж”ҫзҡ„е…¬еҸёеҲӣйҖ ж–°зҡ„ж”¶е…ҘжқҘжәҗгҖӮ

дҫӢеҰӮпјҢдёӯеӣҪеҲӣе»әдәҶдёҖдёӘеҹәдәҺеҢәеқ—й“ҫзҡ„е№іеҸ°пјҢз”ЁдәҺи·ҹиёӘиҪҰиҫҶзҡ„зўіжҺ’ж”ҫйҮҸгҖӮиҜҘе№іеҸ°жҸҗдҫӣе®һж—¶зҡ„жҺ’ж”ҫж•°жҚ®пјҢе№¶е…Ғи®ёе…¬еҸёеҗ‘еҮҸе°‘жҺ’ж”ҫзҡ„е…¬еҸёиҙӯд№°зўідҝЎз”ЁйўқеәҰгҖӮиҝҷдёәеҮҸе°‘дәӨйҖҡйғЁй—Ёзҡ„зўіжҺ’ж”ҫеҲӣйҖ дәҶдёҖдёӘжӣҙеҠ йҖҸжҳҺе’Ңй«ҳж•Ҳзҡ„зі»з»ҹгҖӮ

жҖ»д№ӢпјҢеҢәеқ—й“ҫжҠҖжңҜжңүеҸҜиғҪйҖҡиҝҮеҲӣе»әжӣҙеҠ еҸҜжҢҒз»ӯгҖҒй«ҳж•Ҳе’ҢйҖҸжҳҺзҡ„зі»з»ҹпјҢеҪ»еә•ж”№еҸҳиғҪжәҗе’Ңе…¬з”ЁдәӢдёҡиЎҢдёҡгҖӮд»ҺеҲҶж•Јзҡ„иғҪжәҗзҪ‘еҲ°зўідҝЎз”ЁйўқеәҰе’ҢжҺ’ж”ҫи·ҹиёӘ**пјҢеҸҜиғҪжҖ§жҳҜж— йҷҗзҡ„гҖӮ

еҢәеқ—й“ҫеңЁжІ»зҗҶе’Ңе…¬е…ұжңҚеҠЎдёӯзҡ„еә”з”Ё #

еҢәеқ—й“ҫжҠҖжңҜжӯЈеңЁиҝ…йҖҹж”№еҸҳж”ҝеәңе’Ңе…¬е…ұжңҚеҠЎзҡ„иҝҗдҪңж–№ејҸгҖӮйҖҡиҝҮжҸҗдҫӣдёҖдёӘе®үе…Ёе’ҢеҺ»дёӯеҝғеҢ–зҡ„зі»з»ҹжқҘеӯҳеӮЁе’ҢйӘҢиҜҒж•°жҚ®пјҢеҢәеқ—й“ҫжҠҖжңҜжӯЈеңЁдёәеҗ„иЎҢеҗ„дёҡеёҰжқҘдҝЎд»»гҖҒе®үе…Ёе’ҢйҖҸжҳҺгҖӮжң¬ж–Үе°ҶжҺўи®Ё*еҢәеқ—й“ҫжҠҖжңҜеңЁжІ»зҗҶе’Ңе…¬е…ұжңҚеҠЎдёӯзҡ„дёҖдәӣеә”з”Ёж–№ејҸгҖӮ

иә«д»Ҫз®ЎзҗҶ #

иә«д»Ҫзӣ—зӘғжҳҜеҪ“д»Ҡж•°еӯ—дё–з•ҢдёҖдёӘж—ҘзӣҠдёҘйҮҚзҡ„й—®йўҳгҖӮйҡҸзқҖи¶ҠжқҘи¶ҠеӨҡзҡ„дёӘдәәдҝЎжҒҜиў«еӯҳеӮЁеңЁзҪ‘дёҠпјҢдёӘдәәиә«д»Ҫйқўдёҙиў«зӣ—зҡ„йЈҺйҷ©гҖӮеҢәеқ—й“ҫжҠҖжңҜеҸҜд»ҘйҖҡиҝҮеҲӣе»әдёҖдёӘе®үе…Ёе’ҢеҲҶж•Јзҡ„зі»з»ҹжқҘеӯҳеӮЁе’ҢйӘҢиҜҒиә«д»ҪпјҢд»ҺиҖҢж”№е–„иә«д»Ҫз®ЎзҗҶгҖӮйҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„зі»з»ҹ**пјҢдёӘдәәеҸҜд»ҘжӣҙеҘҪең°жҺ§еҲ¶иҮӘе·ұзҡ„дёӘдәәж•°жҚ®пјҢдҝқжҠӨиҮӘе·ұзҡ„иә«д»ҪдёҚиў«зӣ—з”ЁгҖӮ

дҫӢеҰӮпјҢеҹәдәҺеҢәеқ—й“ҫзҡ„иә«д»Ҫз®ЎзҗҶзі»з»ҹеҸҜд»Ҙи®©дёӘдәәеңЁеҢәеқ—й“ҫдёҠеӯҳеӮЁе…¶дёӘдәәдҝЎжҒҜпјҢеҰӮе§“еҗҚгҖҒеҮәз”ҹж—Ҙжңҹе’Ңең°еқҖгҖӮиҝҷдәӣдҝЎжҒҜйҡҸеҗҺдјҡиў«еҠ еҜҶпјҢе№¶д»ҘеҺ»дёӯеҝғеҢ–зҡ„ж–№ејҸеӯҳеӮЁпјҢдҪҝй»‘е®ўеҮ д№Һж— жі•и®ҝй—®гҖӮеҪ“дёӘдәәйңҖиҰҒйӘҢиҜҒиә«д»Ҫж—¶пјҢд»–д»¬еҸҜд»ҘйҖҡиҝҮжҸҗдҫӣз§ҒдәәеҜҶй’ҘжқҘи§Јй”ҒеҢәеқ—й“ҫдёҠзҡ„еҠ еҜҶдҝЎжҒҜгҖӮиҝҷжҸҗдҫӣдәҶдёҖз§Қе®үе…Ёе’Ңй«ҳж•Ҳзҡ„ж–№ејҸжқҘйӘҢиҜҒиә«д»ҪпјҢиҖҢж— йңҖдёӯеӨ®жңәжһ„гҖӮ

жҠ•зҘЁзі»з»ҹ #

жҠ•зҘЁжҳҜж°‘дё»зҡ„дёҖдёӘеҹәжң¬ж–№йқўпјҢдҪҶдј з»ҹзҡ„жҠ•зҘЁзі»з»ҹеҫҖеҫҖе®№жҳ“еҸ—еҲ°ж¬әиҜҲе’Ңж“ҚзәөгҖӮеҢәеқ—й“ҫжҠҖжңҜеҸҜд»ҘйҖҡиҝҮеҲӣе»әдёҖдёӘйҳІзҜЎж”№е’ҢеҸҜе®Ўи®Ўзҡ„жҠ•зҘЁе’Ңи®ЎзҘЁзі»з»ҹпјҢеўһејәжҠ•зҘЁзі»з»ҹзҡ„е®үе…ЁжҖ§е’ҢйҖҸжҳҺеәҰгҖӮйҖҡиҝҮдҪҝз”ЁеҹәдәҺеҢәеқ—й“ҫзҡ„жҠ•зҘЁзі»з»ҹпјҢж”ҝеәңеҸҜд»ҘзЎ®дҝқйҖүдёҫзҡ„е…¬жӯЈжҖ§пјҢе№¶еўһејәйҖүж°‘зҡ„дҝЎеҝғгҖӮ

дҫӢеҰӮпјҢеҹәдәҺеҢәеқ—й“ҫзҡ„жҠ•зҘЁзі»з»ҹеҸҜд»Ҙи®©йҖүж°‘е®үе…Ёең°е’ҢеҢҝеҗҚең°жҠ•зҘЁгҖӮжҜҸеј йҖүзҘЁйғҪи®°еҪ•еңЁеҢәеқ—й“ҫдёҠпјҢдёҖж—ҰжҠ•зҘЁз»“жқҹпјҢйҖүзҘЁе°ұдјҡиҮӘеҠЁз»ҹи®ЎгҖӮз”ұдәҺеҢәеқ—й“ҫе…·жңүйҳІзҜЎж”№е’ҢеҸҜе®Ўи®Ўзҡ„зү№зӮ№пјҢд»»дҪ•дәәеҮ д№ҺйғҪдёҚеҸҜиғҪж“ҚзәөйҖүдёҫз»“жһңгҖӮиҝҷжҸҗдҫӣдәҶдёҖз§ҚйҖҸжҳҺе’Ң**е®үе…Ёзҡ„йҖүдёҫж–№ејҸпјҢеҸҜд»ҘжҸҗй«ҳе…¬дј—еҜ№ж°‘дё»иҝӣзЁӢзҡ„дҝЎд»»гҖӮ

еңҹең°зҷ»и®°е’Ңзү©дёҡз®ЎзҗҶ #

еңҹең°зҷ»и®°е’Ңиҙўдә§з®ЎзҗҶжҳҜеҢәеқ—й“ҫжҠҖжңҜиғҪдә§з”ҹйҮҚеӨ§еҪұе“Қзҡ„еҸҰдёҖдёӘйўҶеҹҹгҖӮзӣ®еүҚпјҢеңҹең°зҷ»и®°е’Ңиҙўдә§з®ЎзҗҶзі»з»ҹйҖҡеёёйҖҹеәҰзј“ж…ўгҖҒж•ҲзҺҮдҪҺдёӢпјҢиҖҢдё”е®№жҳ“еҮәзҺ°ж¬әиҜҲиЎҢдёәгҖӮеҢәеқ—й“ҫжҠҖжңҜеҸҜд»ҘйҖҡиҝҮеҲӣе»әдёҖдёӘе®үе…ЁгҖҒйҳІзҜЎж”№зҡ„зі»з»ҹжқҘи®°еҪ•иҙўдә§жүҖжңүжқғпјҢд»ҺиҖҢжҸҗй«ҳиҝҷдәӣзі»з»ҹзҡ„ж•ҲзҺҮе’ҢйҖҸжҳҺеәҰгҖӮ

дҫӢеҰӮпјҢеҹәдәҺеҢәеқ—й“ҫзҡ„еңҹең°зҷ»и®°зі»з»ҹеҸҜд»Ҙи®©иҙўдә§жүҖжңүдәәеңЁеҢәеқ—й“ҫдёҠзҷ»и®°е…¶иҙўдә§гҖӮжҜҸеӨ„жҲҝдә§йғҪдјҡиў«еҲҶй…ҚдёҖдёӘзӢ¬зү№зҡ„ж•°еӯ—иә«д»ҪпјҢе№¶и®°еҪ•еңЁеҢәеқ—й“ҫдёҠгҖӮиҝҷе°ұжҸҗдҫӣдәҶдёҖдёӘйҳІзҜЎж”№е’ҢеҸҜе®Ўи®Ўзҡ„жүҖжңүжқғи®°еҪ•пјҢеҸҜд»ҘйҷҚдҪҺж¬әиҜҲйЈҺйҷ©пјҢдёәдёҡдё»жҸҗдҫӣжӣҙеӨ§зҡ„зЎ®е®ҡжҖ§гҖӮжӯӨеӨ–пјҢз”ұдәҺеҢәеқ—й“ҫжҳҜеҺ»дёӯеҝғеҢ–зҡ„пјҢе®ғдёҚеҸ—д»»дҪ•дёҖдёӘе®һдҪ“зҡ„жҺ§еҲ¶пјҢиҝҷеҸҜд»ҘйҷҚдҪҺи…җиҙҘзҡ„йЈҺйҷ©гҖӮ

зЁҺж”¶е’Ңе…¬е…ұиҙўж”ҝ #

еҢәеқ—й“ҫжҠҖжңҜиҝҳиғҪеҜ№зЁҺж”¶е’Ңе…¬е…ұиҙўж”ҝдә§з”ҹйҮҚеӨ§еҪұе“ҚгҖӮзӣ®еүҚпјҢзЁҺж”¶е’Ңе…¬е…ұиҙўж”ҝз®ЎзҗҶзі»з»ҹеҫҖеҫҖеӨҚжқӮиҖҢдҪҺж•ҲпјҢеҸҜиғҪеҜјиҮҙи…җиҙҘе’Ңз®ЎзҗҶдёҚе–„гҖӮеҢәеқ—й“ҫжҠҖжңҜеҸҜд»ҘйҖҡиҝҮеҲӣе»әдёҖдёӘйҖҸжҳҺе’Ңе®үе…Ёзҡ„зі»з»ҹ**жқҘж”¶йӣҶе’Ңз®ЎзҗҶе…¬е…ұиө„йҮ‘пјҢд»ҺиҖҢз®ҖеҢ–зЁҺж”¶е’Ңе…¬е…ұиҙўж”ҝгҖӮ

дҫӢеҰӮпјҢеҹәдәҺеҢәеқ—й“ҫзҡ„ж”¶зЁҺзі»з»ҹеҸҜд»Ҙи®©дёӘдәәе’ҢдјҒдёҡе®үе…Ёй«ҳж•Ҳең°зәізЁҺгҖӮжҜҸз¬”д»ҳж¬ҫйғҪи®°еҪ•еңЁеҢәеқ—й“ҫдёҠпјҢд»ҺиҖҢжҸҗдҫӣйҖҸжҳҺгҖҒеҸҜе®Ўи®Ўзҡ„ж”¶зЁҺи®°еҪ•гҖӮжӯӨеӨ–пјҢз”ұдәҺеҢәеқ—й“ҫжҳҜеҺ»дёӯеҝғеҢ–зҡ„**пјҢе®ғдёҚеҸ—д»»дҪ•дёҖдёӘе®һдҪ“зҡ„жҺ§еҲ¶пјҢиҝҷеҸҜд»ҘйҷҚдҪҺи…җиҙҘзҡ„йЈҺйҷ©гҖӮ

жӯЈеҰӮжҲ‘д»¬жүҖзңӢеҲ°зҡ„пјҢеҢәеқ—й“ҫжҠҖжңҜжңүеҸҜиғҪйҖҡиҝҮе®һзҺ°дҝЎд»»гҖҒе®үе…Ёе’ҢйҖҸжҳҺжқҘеҪ»еә•ж”№еҸҳеҗ„иЎҢеҗ„дёҡгҖӮиҷҪз„¶д»ҚжңүеҸҜжү©еұ•жҖ§е’Ңзӣ‘з®ЎжЎҶжһ¶зӯүжҢ‘жҲҳжңүеҫ…и§ЈеҶіпјҢдҪҶеҲӣж–°е’Ңйў иҰҶзҡ„жңәдјҡжҳҜе·ЁеӨ§зҡ„гҖӮеҗ„еӣҪж”ҝеәңе’Ңе…¬е…ұжңҚеҠЎйғЁй—Ёеә”з»§з»ӯжҺўзҙў***еҢәеқ—й“ҫжҠҖжңҜзҡ„жҪңеҠӣпјҢе№¶жҺҘеҸ—е…¶еҸҳйқ©еҠӣйҮҸгҖӮ
