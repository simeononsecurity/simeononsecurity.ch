LinuxжҳҜдёҖз§ҚеӨҡеҠҹиғҪгҖҒе®үе…Ёзҡ„ж“ҚдҪңзі»з»ҹпјҢз”ұдәҺе…¶ејҖжәҗжҖ§пјҢеңЁзҪ‘з»ңе®үе…ЁиЎҢдёҡеҫ—еҲ°е№ҝжіӣеә”з”ЁгҖӮдёҚиҝҮпјҢеҜ№дәҺеҲқеӯҰиҖ…жқҘиҜҙпјҢдҪҝз”Ё Linux е‘Ҫд»ӨиЎҢз•Ңйқў (CLI) жү§иЎҢзҪ‘з»ңе®үе…Ёд»»еҠЎеҸҜиғҪдјҡд»ӨдәәжңӣиҖҢз”ҹз•ҸгҖӮжң¬жҢҮеҚ—ж—ЁеңЁеҗ‘еҲқеӯҰиҖ…жҰӮиҝ°еҜ№зҪ‘з»ңе®үе…Ёжңүз”Ёзҡ„еҹәжң¬е’Ңй«ҳзә§ Linux CLI е‘Ҫд»ӨгҖӮ

з”ЁдәҺзҪ‘з»ңе®үе…Ёзҡ„еҹәжң¬ Linux е‘Ҫд»Ө #

жү“еҚ°е·ҘдҪңзӣ®еҪ• #

pwdпјҲжү“еҚ°е·ҘдҪңзӣ®еҪ•пјүе‘Ҫд»Өз”ЁдәҺеңЁ CLI дёӯжҳҫзӨәеҪ“еүҚзҡ„е·ҘдҪңзӣ®еҪ•гҖӮе·ҘдҪңзӣ®еҪ•жҳҜеҪ“еүҚжүҖеңЁж–Үд»¶зі»з»ҹзҡ„зӣ®еҪ•гҖӮиҜҘе‘Ҫд»ӨжңүеҠ©дәҺеңЁж–Үд»¶зі»з»ҹдёӯеҜјиҲӘпјҢе№¶дәҶи§ЈиҮӘе·ұдёҺе…¶д»–зӣ®еҪ•зҡ„дҪҚзҪ®е…ізі»гҖӮдҫӢеҰӮпјҢеҰӮжһңжӮЁеңЁ home зӣ®еҪ•дёӯпјҢе№¶жғіеҜјиҲӘеҲ° documents зӣ®еҪ•пјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ pwd
/home/user
$ cd documents
$ pwd
/home/user/documents

еңЁдёҠдҫӢдёӯпјҢз¬¬дёҖжқЎе‘Ҫд»Өжү“еҚ°еҪ“еүҚе·ҘдҪңзӣ®еҪ•пјҢеҚі home зӣ®еҪ•гҖӮз¬¬дәҢжқЎе‘Ҫд»Өе°Ҷзӣ®еҪ•жӣҙж”№дёәж–ҮжЎЈзӣ®еҪ•пјҢз¬¬дёүжқЎе‘Ҫд»Өжү“еҚ°еҪ“еүҚе·ҘдҪңзӣ®еҪ•пјҢеҚізҺ°еңЁзҡ„ж–ҮжЎЈзӣ®еҪ•гҖӮ

еҲ—иЎЁ #

ls е‘Ҫд»Өз”ЁдәҺеңЁ CLI дёӯеҲ—еҮәдёҖдёӘзӣ®еҪ•зҡ„еҶ…е®№гҖӮиҜҘе‘Ҫд»ӨжҳҫзӨәеҪ“еүҚе·ҘдҪңзӣ®еҪ•дёӯж–Үд»¶е’Ңзӣ®еҪ•зҡ„еҗҚз§°гҖӮиҜҘе‘Ҫд»ӨеҜ№дәҺзЎ®е®ҡзү№е®ҡдҪҚзҪ®зҡ„ж–Үд»¶е’Ңзӣ®еҪ•йқһеёёжңүз”ЁгҖӮдҫӢеҰӮпјҢеҰӮжһңиҰҒжҹҘзңӢ documents зӣ®еҪ•зҡ„еҶ…е®№пјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ ls documents
file1.txt file2.pdf file3.docx

еңЁдёҠдҫӢдёӯпјҢиҜҘе‘Ҫд»ӨеҲ—еҮәдәҶ documents зӣ®еҪ•зҡ„еҶ…е®№пјҢе…¶дёӯеҢ…еҗ«дёүдёӘж–Үд»¶пјҡfile1.txtгҖҒfile2.pdf е’Ң file3.docxгҖӮдҪ д№ҹеҸҜд»ҘдёҚеёҰд»»дҪ•еҸӮж•°дҪҝз”ЁиҜҘе‘Ҫд»ӨжқҘеҲ—еҮәеҪ“еүҚе·ҘдҪңзӣ®еҪ•зҡ„еҶ…е®№гҖӮдҫӢеҰӮ

$ ls
Desktop Documents Downloads Music Pictures Public Templates Videos

еңЁдёҠдҫӢдёӯпјҢиҜҘе‘Ҫд»ӨеҲ—еҮәдәҶдё»зӣ®еҪ•зҡ„еҶ…е®№пјҢе…¶дёӯеҢ…еҗ«еӨҡдёӘеӯҗзӣ®еҪ•пјҢеҰӮжЎҢйқўгҖҒж–ҮжЎЈгҖҒдёӢиҪҪзӯүгҖӮ

жӣҙж”№зӣ®еҪ• #

cdпјҲжӣҙж”№зӣ®еҪ•пјүе‘Ҫд»Өз”ЁдәҺжӣҙж”№ CLI дёӯзҡ„еҪ“еүҚе·ҘдҪңзӣ®еҪ•гҖӮдҪҝз”ЁиҜҘе‘Ҫд»ӨеҸҜд»ҘжөҸи§Ҳж–Үд»¶зі»з»ҹе№¶и®ҝй—®дёҚеҗҢдҪҚзҪ®зҡ„ж–Үд»¶гҖӮдҫӢеҰӮпјҢеҰӮжһңиҰҒе°ҶеҪ“еүҚе·ҘдҪңзӣ®еҪ•жӣҙж”№дёәж–ҮжЎЈзӣ®еҪ•пјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ cd documents

еңЁдёҠдҫӢдёӯпјҢиҜҘе‘Ҫд»Өе°ҶеҪ“еүҚе·ҘдҪңзӣ®еҪ•жӣҙж”№дёәдҪҚдәҺеҪ“еүҚе·ҘдҪңзӣ®еҪ•дёӯзҡ„ж–ҮжЎЈзӣ®еҪ•гҖӮд№ҹеҸҜд»ҘдҪҝз”ЁеёҰжңүз»қеҜ№и·Ҝеҫ„жҲ–зӣёеҜ№и·Ҝеҫ„зҡ„е‘Ҫд»ӨпјҢе°Ҷе·ҘдҪңзӣ®еҪ•жӣҙж”№дёәдёҚеңЁеҪ“еүҚе·ҘдҪңзӣ®еҪ•дёӯзҡ„зӣ®еҪ•гҖӮдҫӢеҰӮ

$ cd /usr/local/bin

еңЁдёҠдҫӢдёӯпјҢиҜҘе‘Ҫд»Өе°ҶеҪ“еүҚе·ҘдҪңзӣ®еҪ•жӣҙж”№дёә /usr/local/bin зӣ®еҪ•пјҢиҝҷжҳҜдёҖдёӘз»қеҜ№и·Ҝеҫ„гҖӮжҲ–иҖ…пјҢд№ҹеҸҜд»ҘдҪҝз”ЁзӣёеҜ№и·Ҝеҫ„жқҘжӣҙж”№е·ҘдҪңзӣ®еҪ•гҖӮдҫӢеҰӮ

$ cd ../..

еңЁдёҠдҫӢдёӯпјҢиҜҘе‘Ҫд»Өе°ҶеҪ“еүҚе·ҘдҪңзӣ®еҪ•дёҠз§»дёӨзә§гҖӮз¬ҰеҸ·"… “д»ЈиЎЁзҲ¶зӣ®еҪ•пјҢдҪ еҸҜд»Ҙз”Ёе®ғжқҘеҗ‘дёҠжөҸи§Ҳзӣ®еҪ•ж ‘гҖӮ

иҝһжҺҘ #

catпјҲиҝһжҺҘпјүе‘Ҫд»Өз”ЁдәҺеңЁ CLI дёӯжҳҫзӨәж–Үд»¶еҶ…е®№гҖӮиҜҘе‘Ҫд»ӨеҜ№дәҺжҹҘзңӢж—Ҙеҝ—ж–Үд»¶жҲ–й…ҚзҪ®ж–Үд»¶зӯүж–Үжң¬ж–Үд»¶зҡ„еҶ…е®№йқһеёёжңүз”ЁгҖӮдҫӢеҰӮпјҢеҰӮжһңиҰҒжҹҘзңӢеҗҚдёә “file1.txt “зҡ„ж–Үд»¶еҶ…е®№пјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ cat file1.txt

еңЁдёҠдҫӢдёӯпјҢиҜҘе‘Ҫд»ӨжҳҫзӨәж–Үд»¶ “file1.txt “зҡ„еҶ…е®№гҖӮжӮЁиҝҳеҸҜд»ҘдҪҝз”ЁиҜҘе‘Ҫд»ӨеңЁ CLI дёӯиҝһжҺҘеӨҡдёӘж–Үд»¶е№¶жҳҫзӨәе…¶еҶ…е®№гҖӮдҫӢеҰӮ

$ cat file1.txt file2.txt file3.txt

еңЁдёҠдҫӢдёӯпјҢиҜҘе‘Ҫд»ӨжҳҫзӨәдәҶдёүдёӘж–Үд»¶зҡ„еҶ…е®№пјҡfile1.txtгҖҒfile2.txt е’Ң file3.txtгҖӮжӮЁиҝҳеҸҜд»ҘдҪҝз”ЁиҜҘе‘Ҫд»Өзҡ„йҖҡй…Қз¬ҰжқҘиҝһжҺҘз¬ҰеҗҲзү№е®ҡжЁЎејҸзҡ„жүҖжңүж–Үд»¶гҖӮдҫӢеҰӮ

$ cat *.txt

еңЁдёҠдҫӢдёӯпјҢиҜҘе‘Ҫд»ӨжҳҫзӨәеҪ“еүҚе·ҘдҪңзӣ®еҪ•дёӯжү©еұ•еҗҚдёә”.txt “зҡ„жүҖжңүж–Үд»¶зҡ„еҶ…е®№гҖӮиҜҘе‘Ҫд»ӨеҸҜз”ЁдәҺеҝ«йҖҹжҹҘзңӢеӨҡдёӘж–Үд»¶зҡ„еҶ…е®№пјҢиҖҢж— йңҖйҖҗдёӘжү“ејҖгҖӮ

е…ЁеұҖжӯЈеҲҷиЎЁиҫҫејҸжү“еҚ° #

grepпјҲе…ЁеұҖжӯЈеҲҷиЎЁиҫҫејҸжү“еҚ°пјүе‘Ҫд»Өз”ЁдәҺеңЁ CLI дёӯжҗңзҙўж–Үд»¶жҲ–ж–Үд»¶йӣҶдёӯзҡ„зү№е®ҡеӯ—з¬ҰдёІжҲ–жЁЎејҸгҖӮиҜҘе‘Ҫд»ӨеҜ№дәҺиҜҶеҲ«ж—Ҙеҝ—ж–Үд»¶дёӯзҡ„жЁЎејҸжҲ–жҗңзҙўй…ҚзҪ®ж–Үд»¶дёӯзҡ„зү№е®ҡдҝЎжҒҜйқһеёёжңүз”ЁгҖӮдҫӢеҰӮпјҢеҰӮжһңиҰҒжҗңзҙўеҗҚдёә “logfile.txt “зҡ„ж–Үд»¶дёӯжүҖжңүеҮәзҺ°зҡ„ “error “дёҖиҜҚпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ grep "error" logfile.txt

еңЁдёҠдҫӢдёӯпјҢиҜҘе‘Ҫд»Өжҗңзҙўж–Үд»¶ “logfile.txt “дёӯжүҖжңүеҮәзҺ°зҡ„ “error “дёҖиҜҚпјҢе№¶еңЁ CLI дёӯжҳҫзӨәеҢ№й…Қзҡ„иЎҢгҖӮиҝҳеҸҜд»ҘдҪҝз”ЁжӯЈеҲҷиЎЁиҫҫејҸжҗңзҙўжӣҙеӨҚжқӮзҡ„жЁЎејҸгҖӮдҫӢеҰӮпјҢеҰӮжһңиҰҒжҗңзҙўеҢ…еҗ«д»Ҙ “p “ејҖеӨҙгҖҒ“y “з»“е°ҫзҡ„еҚ•иҜҚзҡ„жүҖжңүиЎҢпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ grep "p.*y" logfile.txt

еңЁдёҠдҫӢдёӯпјҢе‘Ҫд»Өжҗңзҙўж–Үд»¶ “logfile.txt “дёӯеҢ…еҗ«д»Ҙ “p “ејҖеӨҙгҖҒ“y “з»“е°ҫзҡ„жүҖжңүиЎҢгҖӮжӯЈеҲҷиЎЁиҫҫејҸ “p.*y “еҢ№й…Қд»»дҪ•д»Ҙ “p “ејҖеӨҙгҖҒ“y “з»“е°ҫзҡ„еӯ—з¬ҰдёІпјҢдёӯй—ҙеҸҜеҢ…еҗ«д»»ж„Ҹж•°йҮҸзҡ„еӯ—з¬ҰгҖӮиҜҘе‘Ҫд»Өз”ЁдәҺжҹҘжүҫж—Ҙеҝ—ж–Үд»¶дёӯзҡ„жЁЎејҸпјҢжңүеҠ©дәҺиҜҶеҲ«е®үе…ЁеЁҒиғҒжҲ–жҖ§иғҪй—®йўҳгҖӮ

жӣҙж”№жЁЎејҸ #

chmod пјҲжӣҙж”№жЁЎејҸпјүе‘Ҫд»Өз”ЁдәҺеңЁ CLI дёӯжӣҙж”№ж–Үд»¶жҲ–зӣ®еҪ•зҡ„жқғйҷҗгҖӮиҜҘе‘Ҫд»ӨеҜ№дәҺдҝқжҠӨж–Үд»¶е’Ңзӣ®еҪ•зҡ„е®үе…Ёд»ҘеҸҠжҺ§еҲ¶и°ҒеҸҜд»Ҙи®ҝй—®иҝҷдәӣж–Үд»¶е’Ңзӣ®еҪ•иҮіе…ійҮҚиҰҒгҖӮжқғйҷҗз”ұдёүдёӘж•°еӯ—иЎЁзӨәпјҢеҲҶеҲ«еҜ№еә”жүҖжңүиҖ…гҖҒз»„е’Ңе…¶д»–дәәгҖӮиҝҷдәӣж•°еӯ—ж №жҚ®д»ҘдёӢе…¬ејҸи®Ўз®—пјҡ

4 = read
2 = write
1 = execute

дҫӢеҰӮпјҢеҰӮжһңиҰҒдёәеҗҚдёә “file1.txt “зҡ„ж–Үд»¶иөӢдәҲжүҖжңүиҖ…иҜ»еҶҷжқғйҷҗд»ҘеҸҠз»„е’Ңе…¶д»–дәәеҸӘиҜ»жқғйҷҗпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ chmod 644 file1.txt

еңЁдёҠдҫӢдёӯпјҢе‘Ҫд»Өе°Ҷж–Үд»¶ “file1.txt “зҡ„жқғйҷҗи®ҫзҪ®дёә 644гҖӮз¬¬дёҖдёӘж•°еӯ—пјҲ6пјүд»ЈиЎЁжүҖжңүиҖ…зҡ„жқғйҷҗпјҢеҚіиҜ»еҶҷжқғйҷҗпјҲ4 + 2 = 6пјүгҖӮз¬¬дәҢдҪҚж•°еӯ— (4) д»ЈиЎЁз»„зҡ„жқғйҷҗпјҢеҚіеҸӘиҜ»гҖӮз¬¬дёүдҪҚж•°еӯ—пјҲ4пјүд»ЈиЎЁе…¶д»–дәәзҡ„жқғйҷҗпјҢд№ҹжҳҜеҸӘиҜ»гҖӮ

д№ҹеҸҜд»ҘдҪҝз”ЁеёҰеӯ—жҜҚзҡ„е‘Ҫд»ӨжқҘжӣҙж”№жқғйҷҗгҖӮдҫӢеҰӮпјҢеҰӮжһңиҰҒиөӢдәҲж–Үд»¶жүҖжңүиҖ…е’Ңз»„иҜ»еҶҷжқғйҷҗпјҢиҖҢе…¶д»–дәәеҜ№еҗҚдёә “file2.txt “зҡ„ж–Үд»¶жІЎжңүжқғйҷҗпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ chmod ug=rw,o= file2.txt

еңЁдёҠдҫӢдёӯпјҢе‘Ҫд»Өе°Ҷж–Үд»¶ “file2.txt “зҡ„жқғйҷҗи®ҫзҪ®дёә ug=rw,o=пјҢе…¶дёӯ “ug “д»ЈиЎЁжүҖжңүиҖ…е’Ңз»„пјҢ“r “д»ЈиЎЁиҜ»еҸ–жқғйҷҗпјҢ“w “д»ЈиЎЁеҶҷе…ҘжқғйҷҗгҖӮе…¶дёӯпјҢ“ug “д»ЈиЎЁжүҖжңүиҖ…е’Ңз»„пјҢ“r “д»ЈиЎЁиҜ»еҸ–жқғйҷҗпјҢ“w “д»ЈиЎЁеҶҷе…ҘжқғйҷҗгҖӮ"=“з¬ҰеҸ·иЎЁзӨә “е°Ҷжқғйҷҗи®ҫзҪ®дёә”пјҢ“o=“иЎЁзӨә “еҲ йҷӨжүҖжңүд»–дәәжқғйҷҗ”гҖӮиҜҘе‘Ҫд»ӨеҸҜз”ЁдәҺжҺ§еҲ¶ж•Ҹж„ҹж–Үд»¶е’Ңзӣ®еҪ•зҡ„и®ҝй—®жқғйҷҗпјҢйҳІжӯўжңӘз»ҸжҺҲжқғзҡ„и®ҝй—®гҖӮ

зҪ‘з»ңе®үе…Ёзҡ„й«ҳзә§ Linux е‘Ҫд»Ө #

зҪ‘з»ңжҳ е°„еҷЁ #

nmap е‘Ҫд»ӨжҳҜ CLI дёӯдёҖдёӘеҠҹиғҪејәеӨ§зҡ„зҪ‘з»ңжү«жҸҸе·Ҙе…·пјҢеҸҜз”ЁдәҺиҜҶеҲ«зҪ‘з»ңдёҠзҡ„дё»жңәе’ҢжңҚеҠЎд»ҘеҸҠжҪңеңЁжјҸжҙһгҖӮиҜҘе‘Ҫд»ӨеҸҜд»Ҙжү§иЎҢдёҖзі»еҲ—жү«жҸҸжҠҖжңҜпјҢеҢ…жӢ¬дё»жңәеҸ‘зҺ°гҖҒз«ҜеҸЈжү«жҸҸе’Ңж“ҚдҪңзі»з»ҹжЈҖжөӢгҖӮ

nmap жңҖеҹәжң¬зҡ„з”ЁйҖ”д№ӢдёҖжҳҜжү«жҸҸеҚ•дёӘ IP ең°еқҖжҲ–дё»жңәгҖӮдҫӢеҰӮпјҢиҰҒжү«жҸҸеҚ•дёӘ IP ең°еқҖ (192.168.1.1) зҡ„ејҖж”ҫз«ҜеҸЈпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ nmap 192.168.1.1

дёҠиҝ°е‘Ҫд»Өе°ҶеҜ№зӣ®ж Ү IP иҝҗиЎҢ TCP SYN жү«жҸҸпјҢе№¶иҝ”еӣһејҖж”ҫз«ҜеҸЈеҲ—иЎЁгҖӮиҫ“еҮәз»“жһңе°ҶжҳҫзӨәжү“ејҖзҡ„з«ҜеҸЈгҖҒжҜҸдёӘз«ҜеҸЈдёҠиҝҗиЎҢзҡ„жңҚеҠЎгҖҒз«ҜеҸЈзҠ¶жҖҒпјҲжү“ејҖ/е…ій—ӯпјүпјҢжңүж—¶иҝҳдјҡжҳҫзӨәе…¶д»–дҝЎжҒҜпјҢеҰӮзӣ®ж ҮдёҠиҝҗиЎҢзҡ„ж“ҚдҪңзі»з»ҹгҖӮ

Nmap иҝҳеҸҜз”ЁдәҺеҗҢж—¶жү«жҸҸеӨҡдёӘдё»жңәжҲ– IP ең°еқҖгҖӮдҫӢеҰӮпјҢиҰҒжү«жҸҸ IP ең°еқҖиҢғеӣҙпјҲ192.168.1.1-255пјүеҶ…зҡ„ејҖж”ҫз«ҜеҸЈпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ nmap 192.168.1.1-255

дёҠиҝ°е‘Ҫд»Өе°Ҷжү«жҸҸиҢғеӣҙеҶ…зҡ„жүҖжңү IP ең°еқҖпјҢе№¶иҝ”еӣһжҜҸдёӘзӣ®ж Үзҡ„ејҖж”ҫз«ҜеҸЈе’ҢжңҚеҠЎгҖӮ

йҷӨдәҶеҹәжң¬зҡ„дё»жңәеҸ‘зҺ°е’Ңз«ҜеҸЈжү«жҸҸеӨ–пјҢnmap иҝҳиғҪжү§иЎҢжӣҙй«ҳзә§зҡ„жү«жҸҸпјҢеҰӮжңҚеҠЎе’ҢзүҲжң¬жЈҖжөӢгҖҒж“ҚдҪңзі»з»ҹжЈҖжөӢе’ҢжјҸжҙһжү«жҸҸгҖӮиҝҷдәӣжү«жҸҸжңүеҠ©дәҺеҸ‘зҺ°зҪ‘з»ңдёӯжҪңеңЁзҡ„е®үе…ЁжјҸжҙһпјҢзЎ®дҝқзҪ‘з»ңе…ҚеҸ—жҪңеңЁж”»еҮ»гҖӮ

TCP ж•°жҚ®еҢ…жҢ–жҺҳжңә #

tcpdump е‘Ҫд»Өз”ЁдәҺеңЁ CLI дёӯжҚ•иҺ·е’ҢеҲҶжһҗзҪ‘з»ңжөҒйҮҸпјҢжңүеҠ©дәҺжҺ’йҷӨзҪ‘з»ңж•…йҡңгҖҒеҲҶжһҗзҪ‘з»ңиЎҢдёәе’ҢиҜҶеҲ«жҪңеңЁзҡ„е®үе…ЁеЁҒиғҒгҖӮиҜҘе‘Ҫд»ӨеҸҜе®һж—¶жҚ•иҺ·ж•°жҚ®еҢ…пјҢе№¶еҸҜж №жҚ®жәҗжҲ–зӣ®ж Ү IP ең°еқҖгҖҒеҚҸи®®е’Ңз«ҜеҸЈзӯүеҗ„з§ҚжқЎд»¶иҝҮж»Өж•°жҚ®еҢ…гҖӮ

tcpdump жңҖеҹәжң¬зҡ„з”ЁйҖ”д№ӢдёҖжҳҜжҚ•иҺ·зү№е®ҡжҺҘеҸЈдёҠзҡ„жүҖжңүзҪ‘з»ңжөҒйҮҸгҖӮдҫӢеҰӮпјҢиҰҒжҚ•иҺ· eth0 жҺҘеҸЈдёҠзҡ„жүҖжңүжөҒйҮҸпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ sudo tcpdump -i eth0

дёҠиҝ°е‘Ҫд»Өе°ҶжҚ•иҺ· eth0 жҺҘеҸЈдёҠзҡ„жүҖжңүж•°жҚ®еҢ…пјҢе№¶еңЁ CLI дёӯе®һж—¶жҳҫзӨәгҖӮиҫ“еҮәе°ҶжҳҫзӨәжәҗе’Ңзӣ®ж Ү IP ең°еқҖгҖҒеҚҸи®®д»ҘеҸҠжҜҸдёӘж•°жҚ®еҢ…зҡ„е…¶д»–дҝЎжҒҜгҖӮ

Tcpdump иҝҳеҸҜз”ЁдәҺж №жҚ®еҗ„з§ҚжқЎд»¶иҝҮж»Өж•°жҚ®еҢ…гҖӮдҫӢеҰӮпјҢиҰҒжҚ•иҺ·еҸ‘еҫҖжҲ–еҸ‘иҮӘзү№е®ҡ IP ең°еқҖзҡ„жүҖжңүж•°жҚ®еҢ…пјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ sudo tcpdump host 192.168.1.1

дёҠиҝ°е‘Ҫд»Өе°ҶжҚ•иҺ·жүҖжңүеҸ‘йҖҒеҲ°жҲ–жқҘиҮӘ IP ең°еқҖ 192.168.1.1 зҡ„ж•°жҚ®еҢ…пјҢе№¶еңЁ CLI дёӯе®һж—¶жҳҫзӨәгҖӮжӮЁиҝҳеҸҜд»Ҙж №жҚ®еҚҸи®®пјҲеҰӮ TCPгҖҒUDPпјүгҖҒз«ҜеҸЈеҸ·жҲ–е…¶д»–жқЎд»¶иҝҮж»Өж•°жҚ®еҢ…гҖӮ

йҷӨдәҶе®һж—¶жҚ•иҺ·ж•°жҚ®еҢ…еӨ–пјҢtcpdump иҝҳеҸҜз”ЁдәҺе°Ҷж•°жҚ®еҢ…жҚ•иҺ·еҲ°ж–Үд»¶дёӯпјҢд»Ҙдҫӣж—ҘеҗҺеҲҶжһҗгҖӮдҫӢеҰӮпјҢиҰҒжҚ•иҺ· eth0 жҺҘеҸЈдёҠзҡ„жүҖжңүж•°жҚ®еҢ…е№¶е°Ҷе…¶дҝқеӯҳеҲ°еҗҚдёә “capture.pcap “зҡ„ж–Үд»¶дёӯпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ sudo tcpdump -i eth0 -w capture.pcap

дёҠиҝ°е‘Ҫд»Өе°ҶжҚ•иҺ· eth0 жҺҘеҸЈдёҠзҡ„жүҖжңүж•°жҚ®еҢ…пјҢе№¶д»Ҙ pcap ж јејҸдҝқеӯҳеҲ°ж–Үд»¶ “capture.pcap “дёӯпјҢзЁҚеҗҺеҸҜдҪҝз”Ё Wireshark зӯүе·Ҙе…·иҝӣиЎҢеҲҶжһҗгҖӮиҜҘе‘Ҫд»ӨеҸҜз”ЁдәҺеҲҶжһҗзҪ‘з»ңиЎҢдёәе’ҢиҜҶеҲ«жҪңеңЁзҡ„е®үе…ЁеЁҒиғҒгҖӮ

иҝӣзЁӢзҠ¶жҖҒ #

ps е‘Ҫд»ӨеҸҜеңЁ CLI дёӯжҳҫзӨә Linux зі»з»ҹдёҠжӯЈеңЁиҝҗиЎҢзҡ„иҝӣзЁӢдҝЎжҒҜпјҢжңүеҠ©дәҺиҜҶеҲ«зі»з»ҹдёҠеҸҜиғҪжӯЈеңЁиҝҗиЎҢе№¶еҸҜиғҪеҚұеҸҠзі»з»ҹе®үе…Ёзҡ„еҸҜз–‘иҝӣзЁӢгҖӮиҜҘе‘Ҫд»ӨеҸҜд»ҘжҳҫзӨәиҝҗиЎҢдёӯиҝӣзЁӢзҡ„еҗ„з§ҚдҝЎжҒҜпјҢеҢ…жӢ¬иҝӣзЁӢ ID (PID)гҖҒз”ЁжҲ·гҖҒCPU е’ҢеҶ…еӯҳдҪҝз”Ёжғ…еҶөд»ҘеҸҠе‘Ҫд»ӨеҗҚз§°гҖӮ

ps зҡ„жңҖеҹәжң¬з”ЁйҖ”д№ӢдёҖжҳҜжҳҫзӨәзі»з»ҹдёӯжүҖжңүиҝҗиЎҢиҝӣзЁӢзҡ„еҲ—иЎЁгҖӮдҫӢеҰӮпјҢиҰҒжҳҫзӨәзі»з»ҹдёҠиҝҗиЎҢзҡ„жүҖжңүиҝӣзЁӢеҲ—иЎЁпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ ps aux

дёҠиҝ°е‘Ҫд»Өе°ҶжҳҫзӨәзі»з»ҹдёҠжүҖжңүжӯЈеңЁиҝҗиЎҢзҡ„иҝӣзЁӢеҲ—иЎЁпјҢд»ҘеҸҠе®ғд»¬зҡ„ PIDгҖҒз”ЁжҲ·гҖҒCPU е’ҢеҶ…еӯҳдҪҝз”Ёжғ…еҶөд»ҘеҸҠе‘Ҫд»ӨеҗҚз§°гҖӮиҜҘе‘Ҫд»ӨжңүеҠ©дәҺе…ЁйқўдәҶи§Јзі»з»ҹдёҠиҝҗиЎҢзҡ„иҝӣзЁӢпјҢе№¶иҜҶеҲ«еҸҜиғҪжӯЈеңЁиҝҗиЎҢзҡ„д»»дҪ•еҸҜз–‘иҝӣзЁӢгҖӮ

Ps иҝҳеҸҜз”ЁдәҺжҳҫзӨәзү№е®ҡиҝӣзЁӢжҲ–иҝӣзЁӢйӣҶзҡ„дҝЎжҒҜгҖӮдҫӢеҰӮпјҢиҰҒжҳҫзӨәе…·жңүзү№е®ҡ PIDпјҲеҰӮ PID 1234пјүзҡ„иҝӣзЁӢзҡ„дҝЎжҒҜпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ ps -p 1234

дёҠиҝ°е‘Ҫд»Өе°ҶжҳҫзӨә PID дёә 1234 зҡ„иҝӣзЁӢдҝЎжҒҜпјҢеҢ…жӢ¬е…¶з”ЁжҲ·гҖҒCPU е’ҢеҶ…еӯҳдҪҝз”Ёжғ…еҶөд»ҘеҸҠе‘Ҫд»ӨеҗҚз§°гҖӮжӮЁиҝҳеҸҜд»ҘйҖҡиҝҮжҢҮе®ҡеӨҡдёӘ PID жқҘжҳҫзӨәдёҖз»„иҝӣзЁӢзҡ„дҝЎжҒҜгҖӮ

йҷӨдәҶжҳҫзӨәиҝҗиЎҢиҝӣзЁӢзҡ„дҝЎжҒҜеӨ–пјҢps иҝҳеҸҜз”ЁдәҺе®һж—¶зӣ‘жҺ§иҝӣзЁӢзҡ„зҠ¶жҖҒгҖӮдҫӢеҰӮпјҢиҰҒе®һж—¶зӣ‘жҺ§зү№е®ҡиҝӣзЁӢпјҲеҰӮ PID 1234пјүзҡ„ CPU е’ҢеҶ…еӯҳдҪҝз”Ёжғ…еҶөпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ ps -p 1234 -o %cpu,%mem

дёҠиҝ°е‘Ҫд»Өе°Ҷе®һж—¶жҳҫзӨә PID дёә 1234 зҡ„иҝӣзЁӢзҡ„ CPU е’ҢеҶ…еӯҳдҪҝз”Ёжғ…еҶөпјҢжҜҸз§’жӣҙж–°дёҖж¬Ўиҫ“еҮәгҖӮиҜҘе‘Ҫд»ӨеҸҜз”ЁдәҺзӣ‘жҺ§е…ій”®иҝӣзЁӢзҡ„жҖ§иғҪпјҢе№¶жүҫеҮәжҪңеңЁзҡ„жҖ§иғҪз“¶йўҲгҖӮ

зҪ‘з»ңз»ҹи®Ў #

netstat е‘Ҫд»ӨеңЁ CLI дёӯжҳҫзӨә Linux зі»з»ҹдёҠжҙ»еҠЁзҪ‘з»ңиҝһжҺҘзҡ„дҝЎжҒҜпјҢжңүеҠ©дәҺиҜҶеҲ«жңӘз»ҸжҺҲжқғзҡ„зҪ‘з»ңиҝһжҺҘе’ҢжҪңеңЁзҡ„е®үе…ЁеЁҒиғҒгҖӮиҜҘе‘Ҫд»ӨеҸҜд»ҘжҳҫзӨәжҙ»еҠЁзҪ‘з»ңиҝһжҺҘзҡ„еҗ„з§ҚдҝЎжҒҜпјҢеҢ…жӢ¬жң¬ең°е’ҢиҝңзЁӢең°еқҖгҖҒдҪҝз”Ёзҡ„еҚҸи®®пјҲеҰӮ TCPгҖҒUDPпјүд»ҘеҸҠиҝһжҺҘзҠ¶жҖҒгҖӮ

netstat жңҖеҹәжң¬зҡ„з”ЁйҖ”д№ӢдёҖжҳҜжҳҫзӨәзі»з»ҹдёӯжүҖжңүжҙ»еҠЁзҪ‘з»ңиҝһжҺҘзҡ„еҲ—иЎЁгҖӮдҫӢеҰӮпјҢиҰҒжҳҫзӨәжүҖжңүжҙ»еҠЁзҪ‘з»ңиҝһжҺҘзҡ„еҲ—иЎЁпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ netstat -a

дёҠиҝ°е‘Ҫд»Өе°ҶжҳҫзӨәзі»з»ҹдёӯжүҖжңүжҙ»еҠЁзҪ‘з»ңиҝһжҺҘзҡ„еҲ—иЎЁпјҢд»ҘеҸҠе®ғд»¬зҡ„жң¬ең°е’ҢиҝңзЁӢең°еқҖгҖҒдҪҝз”Ёзҡ„еҚҸи®®е’ҢиҝһжҺҘзҠ¶жҖҒгҖӮиҜҘе‘Ҫд»ӨжңүеҠ©дәҺе…ЁйқўдәҶи§Јзі»з»ҹдёӯзҡ„жҙ»еҠЁзҪ‘з»ңиҝһжҺҘпјҢе№¶иҜҶеҲ«д»»дҪ•жңӘз»ҸжҺҲжқғзҡ„иҝһжҺҘгҖӮ

Netstat иҝҳеҸҜз”ЁдәҺжҳҫзӨәзү№е®ҡеҚҸи®®пјҲеҰӮ TCPпјүзҡ„зҪ‘з»ңиҝһжҺҘдҝЎжҒҜгҖӮдҫӢеҰӮпјҢиҰҒжҳҫзӨәзі»з»ҹдёҠжүҖжңүжҙ»еҠЁ TCP иҝһжҺҘзҡ„еҲ—иЎЁпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ netstat -at

дёҠиҝ°е‘Ҫд»Өе°ҶжҳҫзӨәзі»з»ҹдёӯжүҖжңүжҙ»еҠЁ TCP иҝһжҺҘзҡ„еҲ—иЎЁгҖҒжң¬ең°е’ҢиҝңзЁӢең°еқҖд»ҘеҸҠиҝһжҺҘзҠ¶жҖҒгҖӮ

йҷӨдәҶжҳҫзӨәжҙ»еҠЁзҪ‘з»ңиҝһжҺҘзҡ„дҝЎжҒҜеӨ–пјҢnetstat иҝҳеҸҜз”ЁдәҺжҳҫзӨәзү№е®ҡеҚҸи®®пјҲеҰӮ TCPпјүзҡ„зҪ‘з»ңз»ҹи®ЎдҝЎжҒҜгҖӮдҫӢеҰӮпјҢиҰҒжҳҫзӨәзі»з»ҹдёҠжүҖжңү TCP иҝһжҺҘзҡ„з»ҹи®Ўж•°жҚ®пјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ netstat -st

дёҠиҝ°е‘Ҫд»Өе°ҶжҳҫзӨәзі»з»ҹдёҠжүҖжңү TCP иҝһжҺҘзҡ„з»ҹи®ЎдҝЎжҒҜпјҢеҢ…жӢ¬жҙ»еҠЁиҝһжҺҘзҡ„ж•°йҮҸгҖҒеҗ„зҠ¶жҖҒдёӢиҝһжҺҘзҡ„ж•°йҮҸд»ҘеҸҠе·ІеҸ‘з”ҹй”ҷиҜҜзҡ„ж•°йҮҸгҖӮиҜҘе‘Ҫд»ӨеҸҜз”ЁдәҺзӣ‘жҺ§зҪ‘з»ңзҡ„жҖ»дҪ“еҒҘеә·зҠ¶еҶөе№¶жүҫеҮәжҪңеңЁзҡ„жҖ§иғҪй—®йўҳгҖӮ

жҹҘжүҫж–Үд»¶ #

еңЁ CLI дёӯпјҢfind е‘Ҫд»Өз”ЁдәҺжҗңзҙў Linux зі»з»ҹдёӯзҡ„ж–Үд»¶е’Ңзӣ®еҪ•пјҢиҝҷеҜ№жҹҘжүҫеҸҜиғҪйҡҗи—ҸжҲ–йҡҫд»ҘжүҫеҲ°зҡ„зү№е®ҡж–Үд»¶е’Ңзӣ®еҪ•йқһеёёжңүз”ЁгҖӮиҜҘе‘Ҫд»ӨеҸҜж №жҚ®еӨҡз§Қж ҮеҮҶжҗңзҙўж–Үд»¶е’Ңзӣ®еҪ•пјҢеҢ…жӢ¬ж–Үд»¶еҗҚгҖҒеӨ§е°ҸгҖҒдҝ®ж”№ж—¶й—ҙе’ҢжқғйҷҗгҖӮ

жҹҘжүҫзҡ„жңҖеҹәжң¬з”ЁйҖ”д№ӢдёҖжҳҜжҢүеҗҚз§°жҗңзҙўж–Үд»¶е’Ңзӣ®еҪ•гҖӮдҫӢеҰӮпјҢиҰҒжҗңзҙўеҪ“еүҚзӣ®еҪ•еҸҠе…¶еӯҗзӣ®еҪ•дёӯеҗҚз§°дёә “example.txt “зҡ„жүҖжңүж–Үд»¶пјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ find . -name example.txt

дёҠиҝ°е‘Ҫд»Өе°ҶжҗңзҙўеҪ“еүҚзӣ®еҪ•еҸҠе…¶еӯҗзӣ®еҪ•дёӯеҗҚз§°дёә “example.txt “зҡ„жүҖжңүж–Үд»¶пјҢе№¶жҳҫзӨәе…¶е®Ңж•ҙи·Ҝеҫ„гҖӮ

жҹҘжүҫиҝҳеҸҜд»Ҙж №жҚ®ж–Үд»¶е’Ңзӣ®еҪ•зҡ„еӨ§е°ҸиҝӣиЎҢжҗңзҙўгҖӮдҫӢеҰӮпјҢиҰҒжҗңзҙўеҪ“еүҚзӣ®еҪ•еҸҠе…¶еӯҗзӣ®еҪ•дёӯеӨ§дәҺ 1 MB зҡ„жүҖжңүж–Үд»¶пјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ find . -size +1M

дёҠиҝ°е‘Ҫд»Өе°ҶжҗңзҙўеҪ“еүҚзӣ®еҪ•еҸҠе…¶еӯҗзӣ®еҪ•дёӯеӨ§дәҺ 1 MB зҡ„жүҖжңүж–Үд»¶пјҢе№¶жҳҫзӨәе…¶е®Ңж•ҙи·Ҝеҫ„гҖӮ

йҷӨдәҶжҢүеҗҚз§°е’ҢеӨ§е°Ҹжҗңзҙўж–Үд»¶е’Ңзӣ®еҪ•еӨ–пјҢfind иҝҳеҸҜз”ЁдәҺжҢүдҝ®ж”№ж—¶й—ҙе’Ңжқғйҷҗжҗңзҙўж–Үд»¶е’Ңзӣ®еҪ•гҖӮдҫӢеҰӮпјҢиҰҒжҗңзҙўеҪ“еүҚзӣ®еҪ•еҸҠе…¶еӯҗзӣ®еҪ•дёӯеңЁиҝҮеҺ» 7 еӨ©еҶ…дҝ®ж”№иҝҮзҡ„жүҖжңүж–Үд»¶пјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»Өпјҡ

$ find . -mtime -7

дёҠиҝ°е‘Ҫд»Өе°ҶжҗңзҙўеҪ“еүҚзӣ®еҪ•еҸҠе…¶еӯҗзӣ®еҪ•дёӯеңЁиҝҮеҺ» 7 еӨ©еҶ…дҝ®ж”№иҝҮзҡ„жүҖжңүж–Үд»¶пјҢе№¶жҳҫзӨәе…¶е®Ңж•ҙи·Ҝеҫ„гҖӮ

жҖ»д№ӢпјҢfind е‘Ҫд»ӨжҳҜдёҖдёӘеҠҹиғҪејәеӨ§зҡ„е·Ҙе…·пјҢеҸҜж №жҚ®еҗ„з§ҚжқЎд»¶жҗңзҙў Linux зі»з»ҹдёӯзҡ„ж–Үд»¶е’Ңзӣ®еҪ•пјҢеӣ жӯӨеҸҜз”ЁдәҺеҗ„з§Қд»»еҠЎпјҢеҢ…жӢ¬зі»з»ҹз®ЎзҗҶе’ҢзҪ‘з»ңе®үе…ЁгҖӮ

еҜ№дәҺеҲқеӯҰиҖ…жқҘиҜҙпјҢдҪҝз”Ё Linux е‘Ҫд»ӨиЎҢиҝӣиЎҢзҪ‘з»ңе®үе…Ёж“ҚдҪңеҸҜиғҪдјҡд»ӨдәәдёҚзҹҘжүҖжҺӘгҖӮдёҚиҝҮпјҢжңүдәҶжң¬жҢҮеҚ—дёӯжҰӮиҝ°зҡ„еҹәжң¬е’Ңй«ҳзә§е‘Ҫд»ӨпјҢдҪ е°ұеҸҜд»ҘејҖе§ӢдҪҝз”Ё Linux CLIпјҢдҪҝе…¶еңЁзҪ‘з»ңе®үе…Ёж–№йқўеҸ‘жҢҘдјҳеҠҝгҖӮеҲҮи®°еңЁиҝҗиЎҢе‘Ҫд»Өж—¶иҰҒе°Ҹеҝғи°Ёж…ҺпјҢе№¶еңЁдҪҝз”ЁеүҚеҪ»еә•дәҶи§ЈжҜҸдёӘе‘Ҫд»Өзҡ„дҪңз”ЁгҖӮ

иҰҒдәҶи§Јжңүе…ідҪҝз”Ё Linux иҝӣиЎҢзҪ‘з»ңе®үе…Ёзҡ„жӣҙеӨҡдҝЎжҒҜпјҢиҜ·зӮ№еҮ»дёӢиҪҪ ** гҖӮ Kali Linux ж“ҚдҪңзі»з»ҹпјҢдё“й—Ёз”ЁдәҺжё—йҖҸжөӢиҜ•е’Ңж•°еӯ—еҸ–иҜҒгҖӮ

з»“и®ә #

жҖ»д№ӢпјҢLinux е‘Ҫд»ӨиЎҢз•ҢйқўеҜ№дәҺзҪ‘з»ңе®үе…Ёдё“дёҡдәәе‘ҳжқҘиҜҙжҳҜдёҖдёӘејәеӨ§зҡ„е·Ҙе…·пјҢдҪҶеҜ№дәҺеҲқеӯҰиҖ…жқҘиҜҙеҸҜиғҪдјҡжңӣиҖҢз”ҹз•ҸгҖӮйҖҡиҝҮеӯҰд№ жң¬жҢҮеҚ—дёӯжҰӮиҝ°зҡ„еҹәжң¬е‘Ҫд»Өе’Ңй«ҳзә§е‘Ҫд»ӨпјҢдҪ е°ұеҸҜд»ҘејҖе§ӢдҪҝз”Ё Linux CLIпјҢд»ҺиҖҢеңЁзҪ‘з»ңе®үе…ЁйўҶеҹҹеҸ‘жҢҘиҮӘе·ұзҡ„дјҳеҠҝгҖӮиҜ·и®°дҪҸпјҢеңЁиҝҗиЎҢе‘Ҫд»Өж—¶дёҖе®ҡиҰҒе°Ҹеҝғи°Ёж…ҺпјҢе№¶еңЁдҪҝз”ЁеүҚеҪ»еә•дәҶи§ЈжҜҸжқЎе‘Ҫд»Өзҡ„дҪңз”ЁгҖӮйҖҡиҝҮз»ғд№ е’Ңз§ҜзҙҜз»ҸйӘҢпјҢдҪ еҸҜд»ҘзҶҹз»ғдҪҝз”Ё Linux е‘Ҫд»ӨиЎҢпјҢе№¶е°ҶиҮӘе·ұзҡ„зҪ‘з»ңе®үе…ЁжҠҖиғҪжҸҗеҚҮеҲ°дёҖдёӘж–°зҡ„ж°ҙе№ігҖӮ
