–°—В–∞–љ–і–∞—А—В—Л —И–Є—Д—А–Њ–≤–∞–љ–Є—П #

–Ю–±–µ—Б–њ–µ—З–µ–љ–Є–µ –±–µ–Ј–Њ–њ–∞—Б–љ–Њ—Б—В–Є –±–µ—Б–њ—А–Њ–≤–Њ–і–љ—Л—Е —Б–µ—В–µ–є –Є–Љ–µ–µ—В –Њ–≥—А–Њ–Љ–љ–Њ–µ –Ј–љ–∞—З–µ–љ–Є–µ. –°—В–∞–љ–і–∞—А—В—Л —И–Є—Д—А–Њ–≤–∞–љ–Є—П –Є–≥—А–∞—О—В –≤–∞–ґ–љ—Г—О —А–Њ–ї—М –≤ –Ј–∞—Й–Є—В–µ —Б–µ—В–µ–≤–Њ–≥–Њ —В—А–∞—Д–Є–Ї–∞ –Њ—В –љ–µ—Б–∞–љ–Ї—Ж–Є–Њ–љ–Є—А–Њ–≤–∞–љ–љ–Њ–≥–Њ –і–Њ—Б—В—Г–њ–∞. –Ъ —З–Є—Б–ї—Г —И–Є—А–Њ–Ї–Њ –Є—Б–њ–Њ–ї—М–Ј—Г–µ–Љ—Л—Е —Б—В–∞–љ–і–∞—А—В–Њ–≤ —И–Є—Д—А–Њ–≤–∞–љ–Є—П –Њ—В–љ–Њ—Б—П—В—Б—П:

  • WiFi Protected Access (WPA)/WPA2 Personal: –≠—В–Є —Б—В–∞–љ–і–∞—А—В—Л —И–Є—Д—А–Њ–≤–∞–љ–Є—П –Є—Б–њ–Њ–ї—М–Ј—Г—О—В —А–∞—Б—И–Є—А–µ–љ–љ—Л–є —Б—В–∞–љ–і–∞—А—В —И–Є—Д—А–Њ–≤–∞–љ–Є—П (AES) –Є–ї–Є –њ—А–Њ—В–Њ–Ї–Њ–ї –≤—А–µ–Љ–µ–љ–љ–Њ–є —Ж–µ–ї–Њ—Б—В–љ–Њ—Б—В–Є –Ї–ї—О—З–∞ (TKIP) –і–ї—П –Ј–∞—Й–Є—В—Л —Б–µ—В–µ–≤—Л—Е —Б–Њ–µ–і–Є–љ–µ–љ–Є–є –і–ї—П –њ–µ—А—Б–Њ–љ–∞–ї—М–љ–Њ–≥–Њ –Є—Б–њ–Њ–ї—М–Ј–Њ–≤–∞–љ–Є—П.

  • WPA/WPA2 Enterprise: –≠—В–Њ—В —Б—В–∞–љ–і–∞—А—В —И–Є—Д—А–Њ–≤–∞–љ–Є—П, –Є—Б–њ–Њ–ї—М–Ј—Г—О—Й–Є–є AES –Є–ї–Є TKIP, —И–Є—А–Њ–Ї–Њ –њ—А–Є–Љ–µ–љ—П–µ—В—Б—П –≤ –Ї–Њ—А–њ–Њ—А–∞—В–Є–≤–љ—Л—Е —Б—А–µ–і–∞—Е, –Њ–±–µ—Б–њ–µ—З–Є–≤–∞—П –њ–Њ–≤—Л—И–µ–љ–љ—Г—О –±–µ–Ј–Њ–њ–∞—Б–љ–Њ—Б—В—М –Є –Љ–µ—Е–∞–љ–Є–Ј–Љ—Л –∞—Г—В–µ–љ—В–Є—Д–Є–Ї–∞—Ж–Є–Є.

–Ъ–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є SSID #

–Ш–і–µ–љ—В–Є—Д–Є–Ї–∞—В–Њ—А –љ–∞–±–Њ—А–∞ —Г—Б–ї—Г–≥ (SSID) - —Н—В–Њ —Г–љ–Є–Ї–∞–ї—М–љ—Л–є –Є–і–µ–љ—В–Є—Д–Є–Ї–∞—В–Њ—А, –Ї–Њ—В–Њ—А—Л–є –Њ—В–ї–Є—З–∞–µ—В –Њ–і–љ—Г –±–µ—Б–њ—А–Њ–≤–Њ–і–љ—Г—О —Б–µ—В—М –Њ—В –і—А—Г–≥–Њ–є. –°—Г—Й–µ—Б—В–≤—Г—О—В —А–∞–Ј–ї–Є—З–љ—Л–µ –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є SSID, –Њ—В–≤–µ—З–∞—О—Й–Є–µ —Б–њ–µ—Ж–Є—Д–Є—З–µ—Б–Ї–Є–Љ —В—А–µ–±–Њ–≤–∞–љ–Є—П–Љ —Б–µ—В–Є:

  • Basic Service Set (BSS): BSS –њ—А–µ–і—Б—В–∞–≤–ї—П–µ—В —Б–Њ–±–Њ–є –Њ–і–љ—Г —В–Њ—З–Ї—Г –і–Њ—Б—В—Г–њ–∞ (AP) –Є —Б–≤—П–Ј–∞–љ–љ—Л–µ —Б –љ–µ–є –Ї–ї–Є–µ–љ—В—Б–Ї–Є–µ —Г—Б—В—А–Њ–є—Б—В–≤–∞.

  • Extended Service Set (ESS): ESS —Б–Њ—Б—В–Њ–Є—В –Є–Ј –љ–µ—Б–Ї–Њ–ї—М–Ї–Є—Е —В–Њ—З–µ–Ї –і–Њ—Б—В—Г–њ–∞, –Њ–±—К–µ–і–Є–љ–µ–љ–љ—Л—Е –≤ –µ–і–Є–љ—Г—О –ї–Њ–≥–Є—З–µ—Б–Ї—Г—О —Б–µ—В—М.

  • –Э–µ–Ј–∞–≤–Є—Б–Є–Љ—Л–є –љ–∞–±–Њ—А –±–∞–Ј–Њ–≤—Л—Е —Г—Б–ї—Г–≥ (Ad-hoc): –°–µ—В–Є Ad-hoc –њ–Њ–Ј–≤–Њ–ї—П—О—В —Г—Б—В—А–Њ–є—Б—В–≤–∞–Љ —Г—Б—В–∞–љ–∞–≤–ї–Є–≤–∞—В—М –њ—А—П–Љ—Л–µ —Б–Њ–µ–і–Є–љ–µ–љ–Є—П –±–µ–Ј –Є—Б–њ–Њ–ї—М–Ј–Њ–≤–∞–љ–Є—П —В–Њ—З–Ї–Є –і–Њ—Б—В—Г–њ–∞.

  • –†–Њ—Г–Љ–Є–љ–≥: –†–Њ—Г–Љ–Є–љ–≥ –њ–Њ–Ј–≤–Њ–ї—П–µ—В —Г—Б—В—А–Њ–є—Б—В–≤–∞–Љ –њ–ї–∞–≤–љ–Њ –њ–µ—А–µ–Ї–ї—О—З–∞—В—М—Б—П –Љ–µ–ґ–і—Г —В–Њ—З–Ї–∞–Љ–Є –і–Њ—Б—В—Г–њ–∞ –≤ —А–∞–Љ–Ї–∞—Е –Њ–і–љ–Њ–є ESS –±–µ–Ј –њ–Њ—В–µ—А–Є —Б–Њ–µ–і–Є–љ–µ–љ–Є—П.

–Т –Ј–∞–Ї–ї—О—З–µ–љ–Є–µ —Б–ї–µ–і—Г–µ—В –Њ—В–Љ–µ—В–Є—В—М, —З—В–Њ –њ–Њ–љ–Є–Љ–∞–љ–Є–µ —Б—В–∞–љ–і–∞—А—В–Њ–≤ –Є —В–µ—Е–љ–Њ–ї–Њ–≥–Є–є –±–µ—Б–њ—А–Њ–≤–Њ–і–љ–Њ–є —Б–≤—П–Ј–Є —П–≤–ї—П–µ—В—Б—П –ґ–Є–Ј–љ–µ–љ–љ–Њ –≤–∞–ґ–љ—Л–Љ –і–ї—П —Б–µ—В–µ–≤—Л—Е —Б–њ–µ—Ж–Є–∞–ї–Є—Б—В–Њ–≤, —Б—В—А–µ–Љ—П—Й–Є—Е—Б—П –Ї —Г—Б–њ–µ—И–љ–Њ–є —Б–і–∞—З–µ —Б–µ—А—В–Є—Д–Є–Ї–∞—Ж–Є–Њ–љ–љ–Њ–≥–Њ —Н–Ї–Ј–∞–Љ–µ–љ–∞ CompTIA Network+ Certification Exam. –Ш–Ј—Г—З–µ–љ–Є–µ —А–∞–Ј–ї–Є—З–љ—Л—Е —Б—В–∞–љ–і–∞—А—В–Њ–≤ 802.11, —З–∞—Б—В–Њ—В, –і–Є–∞–њ–∞–Ј–Њ–љ–∞, —Б–≤—П–Ј—Л–≤–∞–љ–Є—П –Ї–∞–љ–∞–ї–Њ–≤, —В–Є–њ–Њ–≤ –∞–љ—В–µ–љ–љ, —Б—В–∞–љ–і–∞—А—В–Њ–≤ —И–Є—Д—А–Њ–≤–∞–љ–Є—П –Є –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–є SSID –њ–Њ–Ј–≤–Њ–ї–Є—В —Б–њ–µ—Ж–Є–∞–ї–Є—Б—В–∞–Љ –Ј–∞–ї–Њ–ґ–Є—В—М –њ—А–Њ—З–љ—Л–є —Д—Г–љ–і–∞–Љ–µ–љ—В –њ—А–Є–љ—Ж–Є–њ–Њ–≤ –њ–Њ—Б—В—А–Њ–µ–љ–Є—П –±–µ—Б–њ—А–Њ–≤–Њ–і–љ—Л—Е —Б–µ—В–µ–є.

–Э–µ –Ј–∞–±—Л–≤–∞–є—В–µ —Б–ї–µ–і–Є—В—М –Ј–∞ –њ–Њ—Б–ї–µ–і–љ–Є–Љ–Є –і–Њ—Б—В–Є–ґ–µ–љ–Є—П–Љ–Є –Є –≥–Њ—Б—Г–і–∞—А—Б—В–≤–µ–љ–љ—Л–Љ–Є –љ–Њ—А–Љ–∞–Љ–Є, –Ї–∞—Б–∞—О—Й–Є–Љ–Є—Б—П –±–µ—Б–њ—А–Њ–≤–Њ–і–љ—Л—Е —В–µ—Е–љ–Њ–ї–Њ–≥–Є–є, –њ–Њ—Б–Ї–Њ–ї—М–Ї—Г –Њ–љ–Є –±—Л—Б—В—А–Њ —А–∞–Ј–≤–Є–≤–∞—О—В—Б—П. –Ю—Б–≤–Њ–Є–≤ —Н—В–Є –Ї–Њ–љ—Ж–µ–њ—Ж–Є–Є, –≤—Л –±—Г–і–µ—В–µ —Е–Њ—А–Њ—И–Њ –њ–Њ–і–≥–Њ—В–Њ–≤–ї–µ–љ—Л –Ї —Н—Д—Д–µ–Ї—В–Є–≤–љ–Њ–Љ—Г –њ—А–Њ–µ–Ї—В–Є—А–Њ–≤–∞–љ–Є—О, —А–∞–Ј–≤–µ—А—В—Л–≤–∞–љ–Є—О –Є —Г—Б—В—А–∞–љ–µ–љ–Є—О –љ–µ–Є—Б–њ—А–∞–≤–љ–Њ—Б—В–µ–є –≤ –±–µ—Б–њ—А–Њ–≤–Њ–і–љ—Л—Е —Б–µ—В—П—Е.
