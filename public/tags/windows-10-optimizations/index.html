#Automation  #Windows Updates  #Windows 10 Optimizations  #Windows 10 Debloat  #Windows 10  #Powershell  #Script 
Windows-Optimize-Debloat For those who seek to minimize their Windows 10 installs. Download all the required files from the GitHub Repository Introduction: Windows 10 is an invasive and insecure operating system out of the box. Organizations like Microsoft, PrivacyTools.io, and others have recommended configuration changes to optimize and debloat the Windows 10 operating system. These changes are include blocking telemetry, deleting logs, and removing bloatware to name a few. This script aims to automate the configurations recommended by those organizations.
