CVE-2020-1350 KB4569509 DNS Server Vulnerability

2020-07-25
#CVE-2020-1350  #KB4569509  #DNS Server Vulnerability  #Windows Server  #Powershell  #Script 
KB4569509: Guidance for DNS Server Vulnerability CVE-2020-1350 Introduction On July 14, 2020, Microsoft released a security update for the issue that is described in CVE-2020-1350 | Windows DNS Server Remote Code Execution Vulnerability. This advisory describes a Critical Remote Code Execution (RCE) vulnerability that affects Windows servers that are configured to run the DNS Server role. We strongly recommend that server administrators apply the security update at their earliest convenience.
