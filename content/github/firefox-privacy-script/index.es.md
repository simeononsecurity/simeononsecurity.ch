 Script de privacidad de FireFox  Organizaciones como [PrivacyTools.io](https://www.privacytools.io/browsers/#about_config) y [ffprofile](https://ffprofile.com/) han propuesto cambios para hacer que FireFox sea más seguro y privado. Estos cambios incluyen extensiones de navegador sugeridas, bloqueo de telemetría, deshabilitación de cookies de terceros, deshabilitación de rastreadores, etc.  Este script tomará los archivos de configuración pregenerados de FireFox y los instalará en los directorios adecuados en un sistema Windows.  ## Descarga los archivos requeridos  Descargue los archivos necesarios del [repositorio de GitHub] (https://github.com/simeononsecurity/FireFox-Privacy-Script)  ##ejecutar cómo el script  **El script se puede iniciar desde la descarga de GitHub extraída de esta manera:**
