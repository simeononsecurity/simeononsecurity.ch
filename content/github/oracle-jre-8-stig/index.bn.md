ওরাকল জেআরই এসটিআইজিগুলি এত সোজা নয়, প্রশাসকদের JAVA ডকুমেন্টেশন গবেষণা করতে এবং জাভা কনফিগার ফাইল তৈরি করতে হয়, যখন বেশিরভাগ প্রশাসক শুধুমাত্র গ্রুপ নীতি ব্যবহার করে STIG-ing করতে অভ্যস্ত হয়।
