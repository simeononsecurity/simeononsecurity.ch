用于协助 Windows 组策略部署的 WMI 过滤器集合

在部署 GPO 时，Microsoft 建议使用 WMI 筛选器来加快进程并隔离大型域中的策略。
WMI 筛选器允许您将 GPO 指定到特定的软件或 Windows 版本，并有可能变得更加复杂。
链接的 GitHub 存储库包括一些最常用的 WMI 过滤器，因此您不必创建它们。
