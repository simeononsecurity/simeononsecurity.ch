 ## ما هو Standalone-Windows-STIG-Script؟
 
 ** Standalone-Windows-STIG-Script ** هو نص برمجي تم تطويره بواسطة Simeononsecurity المصمم لأتمتة عملية جعل نظام Windows آمنًا ومتوافقًا مع دليل التنفيذ الفني للأمان (STIG). البرنامج المعروض ومصدره على ** GitHub **.
 
 ## كيف يعمل؟
