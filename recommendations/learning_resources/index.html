一般信息技术： #

如果你想提高自己在通用信息技术方面的技能，有很多选择可供你探索。以下是强烈推荐的几种选择：

Cybrary 这是一个很棒的网络和信息技术学习资源，大部分是免费的，如果你是该领域的新手，这是一个很好的起点。

  • LinkedIn Learning LinkedIn Learning 是一个普通的学习平台，提供包括 IT 课程在内的各种课程。请向您的学校咨询，因为他们可能会提供免费会员资格。
  • Pluralsight Pluralsight 是另一个拥有高质量视频内容的通用学习平台。请向您的学校咨询，因为他们可能会提供免费会员资格。
  • Udemy Udemy 是一个综合性学习平台，提供包括 IT 课程在内的各种课程。

网络安全： #

对于那些对网络安全感兴趣的人，有几种资源可以帮助您学习和获得实践经验。以下是强烈推荐的几个资源：

  • CyberWarfare 该平台提供红队和紫队实验室和认证，帮助您提高技能。
  • HackTheBox 该平台提供黑盒 CTF 挑战赛，帮助您获得实践经验。请注意，它有付费会员选项。
  • INE INE 提供免费和付费的高质量网络安全实验室和培训。
  • Immersive Labs 该平台提供免费的网络安全实验室，也提供付费服务。对在校学生免费。
  • Mosse Cyber Security Institute 该平台提供免费的网络安全实验室，也提供付费服务。
  • PenTesterLab This platform offers hands-on Pentesting Labs 还提供付费服务。
  • TryHackMe 该平台提供有指导的 CTF 挑战，并提供付费会员选项。

网络： #

如果你对网络感兴趣，这里有一些资源可以帮助你发展技能：

  • Cisco 您可以通过思科网络学院免费学习思科课程。请注意，有些课程是收费的。
  • Juniper 您可以通过瞻博网络的学习门户免费学习瞻博网络课程。完成课程即可获得免费认证测试券！

操作系统： #

如果您想进一步了解操作系统，这里有一些资源可以帮助您：

  • Microsoft Microsoft 提供各种主题的免费课程，包括 Windows、服务器、Azure 等！
  • VMWare VMWare 提供有关 VMware ESXi、Horizon 和云计算技术的免费课程，帮助您提高技能。
  • RedHat RedHat 提供有关 RHEL 系统、服务器和 Ansible 的培训。

编程和脚本： #

如果你有兴趣进一步了解编程和脚本，这里有一些资源可供参考：

  • Code.org Code.org 提供免费的编码课程和项目创意，可以帮助您入门。
  • CodeAcademy CodeAcademy 免费提供各种课程，包括 Python、C#、C++、HTML、JavaScript、PHP 等。
  • Python Institute Python 学院提供免费的 Python 入门级课程，帮助您提高技能。

无论您对 IT、网络安全、网络、操作系统还是编程和脚本感兴趣，都有大量资源可以帮助您学习和发展技能。通过 SimeonOnSecurity 的推荐，您可以找到最好的资源来满足您的学习目标，使您的职业生涯更上一层楼。
